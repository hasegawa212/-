# 施工記録ブック作成ツール（construction-book）

株式会社 Martial Arts（不動産・建設）向けの **施工記録ブック（施工ブック）作成ツール**。
工事ごとに「表紙・物件情報 → 工事概要 → 施工工程（写真ページ）→ 完了報告」を入力すると、
そのまま **A4 で印刷 / PDF 出力してお客様にお渡しできる施工記録ブック** が組み上がります。

`echo-interview-console` や `arr-gallery-pamphlet` と同じく **ゼロ依存・単一 HTML** 構成です（ビルド・インストール不要）。

## 特徴

- **左：入力フォーム / 右：A4 ライブプレビュー** の2ペイン構成。入力すると即座に紙面に反映。
- **施工工程ページ**は何ページでも追加可能。各工程に工程名・実施日・説明・複数写真を登録でき、
  写真には **施工前 / 施工中 / 施工後** のラベルとキャプションを付けられます。並び替え・削除も可。
- 写真はアップロード時に **canvas で最大1280px・JPEG圧縮** してから保存するため、localStorage を圧迫しにくい。
- **複数ブックを管理**（「一覧」から保存済みブックの切替・削除）。入力は自動保存（localStorage）。
- **印刷 / PDF 出力**：各セクションが A4 1ページに割り付けられ、ブラウザの印刷ダイアログから
  そのまま印刷、または「PDF として保存」で配布用ファイルになります。
- UI・帳票はすべて日本語。Martial Arts のゴールド／ダークのブランドカラーに準拠。

## 使い方

ビルド不要。静的に配信するだけです。

```bash
cd construction-book
python3 -m http.server 5181   # → http://localhost:5181
```

または `index.html` をブラウザで直接開いても動作します（すべてクライアント側で完結）。

1. 「表紙・物件情報」「工事概要」を入力
2. 「施工工程」で工程ページを追加し、写真を登録（＋ラベル・キャプション）
3. 「完了報告」「会社情報」を入力
4. 右上の **印刷 / PDF** で出力（PDF にするには印刷ダイアログで「PDF として保存」を選択）

## データの保存場所

- すべて **ブラウザの localStorage** に保存されます（サーバー送信なし）。
  - ブック本体：`martialarts_construction_books_v1`
  - 開いていたブックID：`martialarts_construction_active_v1`
- 別の端末・ブラウザには引き継がれません。バックアップが必要な場合は PDF 出力を保管してください。
- 写真を大量に登録すると localStorage の容量上限（概ね 5MB 程度）に達することがあります。
  その場合は写真を減らすか、ブックを分けてください（保存失敗時はステータスに表示されます）。

## カスタマイズ

- 写真ラベルの種類は `index.html` 内の `TAGS` 配列で定義（`before` / `during` / `after` / `none`）。
- 会社情報の初期値は `newBook()` の `company` で設定。
- 画像の圧縮サイズ・画質は `compressImage()` の `MAX`（1280）と JPEG 品質（0.78）で調整。
- 紙面デザインは `@media print` と `.page` 系のスタイルにまとまっています（A4 = 794×1123px @96dpi）。

ビルド・テスト・Lint はありません（単一 HTML）。
