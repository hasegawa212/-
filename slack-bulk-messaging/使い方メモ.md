# Slack 個別DM 一斉送信ツール ／ 使い方メモ

Slackで**宛先リストの一人ひとりに「個別DM」を一斉送信**するツールです。
グループDMではなく1対1で届くので、各人が「自分宛の指示」として受け取ります。

---

## 🚀 使い方（毎回これだけ）

**最初に1回：** ターミナルを開いてフォルダに移動
```
cd ~/slack-tools/slack-bulk-messaging
```

**本番の4ステップ：**

| 手順 | コマンド |
|---|---|
| ① 相手リストを編集 | `open -e recipients.csv` |
| ② 文章を編集 | `open -e message.txt` |
| ③ まず確認（送らない） | `node send.js -r recipients.csv -m message.txt --dry-run` |
| ④ 本番送信 | `node send.js -r recipients.csv -m message.txt` |

### 相手リスト（recipients.csv）の書き方 … 1人1行
```
name,email
田中,tanaka@example.com
鈴木,suzuki@example.com
```
- メールではなくSlack ID（`U...`）で指定する場合は `name,slack_id` の形でもOK

---

## ⚠️ 大事なポイント
- **送信前は必ず ③ `--dry-run`** で内容チェック（誤爆防止／送信は取り消せない）
- 文章の **`{{name}}`** が各人の名前に変わる
- メールは **相手がSlackで使っているもの**（違うと届かない）
- 人数が多い時は末尾に **`--delay 2000`**（送信間隔を2秒に）
- 送信結果は毎回 **`send-log-*.csv`** に記録される

---

## 🛠 困ったときは
| エラー表示 | 対処 |
|---|---|
| `libxcrun ... x86_64` | gitを使う時だけ頭に `arch -arm64 /usr/bin/git ...` を付ける |
| `missing_scope` | Slackアプリの権限不足。権限を足して「Reinstall to Workspace」 |
| `users_not_found` | メールがSlackアカウントと不一致。正しいメールに直す |
| `invalid_auth` | トークンが違う。`open -e .env` で開いて貼り直す |

---

## 🔧 初期設定（済み・参考用）
1. git を使えるようにした（Apple Silicon：git使用時は `arch -arm64 /usr/bin/git ...`）
2. コードをMacにダウンロード（`~/slack-tools`）
3. Slackアプリ（Bot）を作成し、トークンを `.env` に設定
4. 必要な権限（Bot Token Scopes）：`chat:write` / `im:write` / `users:read.email`
   - ※権限を足したら必ず「Reinstall to Workspace」で再インストール
5. 自分宛テストDMで動作確認

### Slackトークンの取得場所
https://api.slack.com/apps → 対象アプリ → OAuth & Permissions →
「Bot User OAuth Token」（`xoxb-` で始まる）を `.env` の `SLACK_BOT_TOKEN=` に貼る
