# 本物査定アプリ（appraisal-app）

不動産（戸建・マンション・土地）と自動車の **概算査定額をその場で算出** するブラウザアプリです。
査定額はモックの固定値ではなく、面積・築年数・構造・走行距離などの入力から
**透明な計算ロジック（査定エンジン）** で算出され、計算の内訳まで表示します。

Vite + React 18 + TypeScript（strict）製の独立したサブプロジェクトで、バックエンド・APIキーは不要です。

## セットアップ

```bash
cd appraisal-app
npm install
npm run dev        # → http://localhost:5175
```

| コマンド            | 内容                                  |
| ------------------- | ------------------------------------- |
| `npm run dev`       | 開発サーバー（Vite, ポート5175）      |
| `npm run build`     | 本番ビルド（`dist/` を出力）          |
| `npm run preview`   | ビルド成果物のプレビュー              |
| `npm run typecheck` | 型チェック（`tsc --noEmit`）          |
| `npm run test`      | 査定エンジンの単体テスト（Vitest）    |

## 画面

ヘッダー下のトグルで **不動産査定 / 車査定 / 投資利回り** を切り替えます。
左に入力フォーム、右に結果（査定額・想定レンジ・計算内訳・注意書き）が表示されます。

「不動産査定」タブはさらに **単一査定 / 一括査定（マイソク）** を切り替えられます。

- **単一査定**：1物件を入力して概算額・想定レンジ・計算内訳を表示。下部に首都圏相場事例との照合も表示。
- **一括査定**：複数物件をまとめて査定し、**提案価格 vs モデル査定額**の乖離から
  「割安／適正／割高」を一覧表で判定します。エンジンは `src/lib/valuation/batch.ts`。入力は2方式：
  - **表形式**：1行1物件・タブ/カンマ区切り（列順：名称, 所在地, 土地面積, 建物面積, 築年数, 構造, 駅徒歩, 価格）。
  - **マイソク本文**：マイソクの本文を貼り付け、**空行で物件を区切る**だけ。「所在地：」「価格：」
    「土地面積：」「建物面積／専有面積：」「築年月：」「構造：」「交通：」などのラベルを自動抽出します。
    表記ゆれにも対応：**坪⇄㎡**、**億/万円**、**和暦（平成・令和）の築年月**、**交通の「徒歩◯分」抽出**。
  - 所在地は自由記述から市区町村キーに自動解決、建物面積0は土地として査定。

「投資利回り」タブは、物件価格・エリア（東京23区）・専有面積から **SUUMO連携の賃料相場
（`wardRents.ts`）で想定家賃を推定**し、表面利回り・実質利回り・月次キャッシュフローを算出して
◎/○/✕ で投資妙味を判定します（家賃を直接入力すれば実額ベースに切替）。エンジンは
`src/lib/valuation/investment.ts`、相場データの更新は `tools/`（SUUMOパイプライン）。

## 査定ロジック

査定エンジンは UI から独立した純粋関数として `src/lib/valuation/` にまとまっており、
単体テスト（`valuation.test.ts`）で挙動を検証しています。

### 不動産（`realEstate.ts`）

- **戸建** = 土地価格（土地面積 × エリア地価 × 駅距離補正）＋ 建物価格（建物面積 × 構造別再調達単価 × 残存率）
- **土地** = 土地価格のみ
- **マンション** = 専有面積 × エリア相場単価 × 残存率 × 駅距離補正
- 残存率は定額法（下限10%）、駅距離補正は徒歩分数に応じた係数
- 査定額の ±8% を想定レンジとして提示

エリア地価は茨城県・栃木県の主要市町村に加え、首都圏（東京都・神奈川県・埼玉県・千葉県）の
代表エリアについて、地価公示の住宅地水準をもとにした概算値（`data.ts` の `CITY_LAND_PRICE`）
を使用しています。所在地のプルダウンは都県ごとにグループ表示されます。

### 自動車（`car.ts`）

```
査定額 = 新車価格 × 年式残価率 × 走行距離補正 × メーカー補正
         ×（修復歴あり 0.6）＋ 車検残ボーナス
```

- 年式残価率は中古車相場の下落カーブ（`AGE_RESIDUAL_CURVE`）を線形補間
- 走行距離補正は「年1万km」を基準に過走行で減点（係数0.6〜1.25でクランプ）
- メーカー補正はリセールバリューの傾向（`MAKER_FACTOR`）を反映
- 下限は新車価格の3%（部品・スクラップ価値）
- 査定額の ±10% を想定レンジとして提示

## カスタマイズ

- **対応エリアを増やす** → `src/lib/valuation/data.ts` の `CITY_LAND_PRICE` に市町村と㎡単価を追加
- **車種プリセットを増やす** → 同 `CAR_MODELS` に `{ maker, model, newPrice }` を追加
- **単価・係数の調整** → `data.ts` の各テーブルを編集（ロジック本体は変更不要）

## 注意

本アプリの査定額は公開データに基づく **概算シミュレーション** であり、実際の売買・買取価格を
保証するものではありません。正式な査定は専門業者の現地調査が必要です。
