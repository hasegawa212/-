# 経営管理

会社経営に関するドキュメント・データを一元管理するためのフォルダです。

## 構成

| フォルダ | 用途 |
| --- | --- |
| `01_経営戦略/` | 中長期計画、年度計画、KPI、取締役会資料など経営方針に関するもの |
| `02_財務・会計/` | 決算、資金繰り、請求書、経費精算、税務 |
| `03_営業/` | 顧客リスト、商談・案件、見積/提案、売上、テレアポ |
| `04_マーケティング/` | キャンペーン、広告、分析、コンテンツ |
| `05_人事・労務/` | 社員名簿、採用、給与、勤怠、評価・教育 |
| `06_顧客管理/` | 顧客情報、契約書、サポート履歴 |
| `07_業務管理/` | 議事録、業務マニュアル、プロジェクト、社内通達 |
| `08_法務・コンプライアンス/` | 契約書雛形、社内規程、知的財産、個人情報保護 |
| `09_資料・アーカイブ/` | テンプレート、過去資料 |

## 運用ルール

- ファイル名の先頭に日付を入れる: `YYYY-MM-DD_概要.ext`（例: `2026-05-11_月次決算.xlsx`）
- 個人情報・機密情報を含むファイルは原則このリポジトリにコミットしない。やむを得ない場合は社内ガイドラインに従い、最低限の範囲に限定する
- 既存のフォルダで分類が難しい場合は、安易に新規フォルダを作らず、まず該当する番号配下にサブフォルダを切る
- アーカイブ対象の古い資料は `09_資料・アーカイブ/過去資料/` に年度ごとに移動する

## 関連ファイル

- `テレアポ管理シート.csv`（リポジトリ直下）: 既存のテレアポ管理データ。今後の運用は `03_営業/テレアポ/` 配下に集約する想定。
