# 役員ルールブック 読み合わせ・運用ガイド

**株式会社Martial Arts｜共通認識と実践的な組織運営のために**（2025年7月14日）

このガイドは、役員間で共通の認識を持ち、より実践的に組織を運営していくための運用手順をまとめたものである。

## 読み合わせの目的

1. 役員間でルールブックの**認識を統一**し、それぞれの**責任を明確化**する
2. 実際に運用する中で生じる**課題や疑問点を事前に洗い出す**
3. 会社の**方向性や価値観を再確認**し、浸透させる
4. 理論だけでなく、**実践的な運用ルールを具体的に策定**する

## スケジュール案

### Phase 1：事前準備（1週間）

対象：全役員・部門責任者

- ルールブック全文の個人熟読
- 担当領域の疑問・提案事項の整理
- 現在の運用との相違点の洗い出し

この事前準備が、集中読み合わせの質を大きく左右する。

### Phase 2：集中読み合わせ（2日間）

#### 1日目：企業理念・組織運営編（3時間）

| 時間 | 内容 |
| :-- | :-- |
| 10:00-10:30 | 全体説明・目的共有（策定背景、TARGET 2030への位置づけ） |
| 10:30-11:30 | 企業理念・行動指針の確認（12の行動指針の実践、部門別の体現方法、具体事例共有） |
| 11:45-12:45 | 会議体制と意思決定プロセス（新会議体制、各役員の役割、報連相・緊急案件フロー演習） |
| 12:45-13:00 | まとめ・宿題設定 |

#### 2日目：実務運用編（3時間）

| 時間 | 内容 |
| :-- | :-- |
| 10:00-10:30 | 前日振り返り・質疑応答 |
| 10:30-11:15 | コンプライアンス・労務管理（ハラスメント防止、労働条件、法令遵守） |
| 11:30-12:15 | 人材育成・ステークホルダー関係（部門別人材育成、顧客・従業員関係、TARGET 2030） |
| 12:15-13:00 | 継続的改善・運用開始（見直しスケジュール、評価方法、課題解決） |

### Phase 3：フォローアップ（1ヶ月後）

役員会議内で30分を確保し、以下を実施する。これはルールブックが「生きている」ことを保証するために重要。

- 運用状況の確認
- 修正・改善点の検討
- 次回見直しスケジュールの確認

## 読み合わせの進行方法

### 基本原則

- **全員参加**：CEO含む全役員の参加必須
- **双方向**：一方的な説明ではなく、議論と確認
- **実践重視**：理論より実際の運用方法に焦点
- **記録**：質疑応答・修正点を詳細に記録

### セクション別の進行方式

| セクション | 進行方式 | ポイント |
| :-- | :-- | :-- |
| 企業理念・行動指針 | ディスカッション | 実践例発表、部門別体現、TARGET 2030コミット |
| 会議体制 | シミュレーション | 意思決定フロー演習、緊急案件訓練 |
| 実務セクション | Q&A＋事例検討 | 特殊事情、問題事例対応、法令変更プロセス |

## 読み合わせ時のチェックポイント

### 理解度確認項目（共通）

- [ ] 企業理念・行動指針の理解と実践
- [ ] 新会議体制と自身の役割
- [ ] 意思決定フロー・緊急時ルート
- [ ] コンプライアンス要求事項の理解

### 実践準備確認項目

- [ ] 各会議の開催準備
- [ ] 部下への説明・指導準備
- [ ] 緊急時対応の準備
- [ ] 定期評価・改善活動の準備

## 読み合わせ後の運用開始

### 段階的運用開始

- **Week 1**：試行運用（新会議体制試行、行動指針実践、問題記録）
- **Week 2-4**：本格運用（全制度運用開始、週次振り返り、社員への浸透）
- **Month 2**：初回評価（役員会議で評価、修正、フィードバック収集）

### 継続的改善サイクル

- **月次**：役員会議で状況確認、小改善
- **四半期**：詳細な効果測定、制度・プロセス見直し
- **年次**：ルールブック全体見直し、TARGET 2030評価

## 成功のためのポイント

- **事前準備の徹底** — 事前熟読と具体的な準備が成功の鍵
- **実践的な議論** — 抽象論ではなく、実務への適用と想定問題への対処を具体的に議論する
- **コミットメントの明確化** — 自身の責任範囲と具体的アクションを明確にし、フォロー方法も確認する
- **継続的な改善** — 完璧を求めず、常に改善前提でスタートし、定期見直しと現場の声を反映する
