> **このレポートについて**

## 0. 要約：AIが評価する営業とは

`sales-analysis` AIを横断分析した結論。**AIは「気合い」も「人柄」も評価しない。AIが見ているのは次の3つだけ。**

1. **数字が文面で固まっているか** — 年収・家計・貯蓄・総支出の上限が "口頭" ではなく "証憑/文面" で確定しているか。AIの口癖は「**信頼は数字で**」「**安心を数字で作る**」。
2. **次が確定し、歩留まりが設計されているか** — 次回アポ確定＋Wリマインド＋事前回収＋同行者手配まで段取りされているか。「**準備8割**」。
3. **進める/見切るの判断が正しいか** — GO/STOP基準を満たすか。動機ゼロ客を深追いせず "丁寧な出口" を作れているか。AIは**正しい打ち切りを明確に称賛する**。

つまり **「数字 × 段取り × 正しい見切り」** が、AIに高評価される営業の方程式です。

## 1. AIボット分析のフォーマット（毎回固定の7ブロック）

`sales-analysis` botは、報告投稿を受けるとまず「生成中.....」を返し、続いて**完全固定の見出し構成**で分析を返します。冒頭は必ず担当営業へのメンション。

| ブロック | 中身 |
|---|---|
| `【要点サマリ】` | 顧客 / 返済余力 / 資産・家計 / ニーズ / 進捗 / **未了**（＝ヒアリング漏れ） |
| `【傾向分析（他顧客含めた傾向）】` | 同型顧客の統計的傾向（年収帯別の勝ちレンジ等） |
| `【課題】` | この案件の減点・リスク |
| `【資金設計（ラフ試算の目安）】` | 借入概算・DTI・固定費の定量試算 |
| `【物件方針（候補の方向性）】` | 価格帯・タイプ・立地・除外条件 |
| `【次の一手（具体アクション）】` | 時間区切り（本日〜48h／〜日付）の打ち手 |
| `【ひと言】` | 励まし・締め |

> **学び**: この7ブロックは、そのまま **あなたが商談後に自問すべきチェック項目**です。「未了は何か」「資金設計は描けるか」「次の一手は48h以内に置けるか」を自分で埋められれば、AIに指摘される前に勝てます。

## 2. AIの定量エンジン（全案件共通の計算式）★最重要

AIは案件ごとに気分で判断していません。**全案件に同じ係数・しきい値**を当てています。これを覚えれば、あなたも"歩くsales-analysis"になれます。

### 2-1. 返済余力（必ず3段階で出す）
```
税込月収 = 年収 ÷ 12
返済負担率 20% / 25% / 30% の3本を提示
例）月収25.6万 → 20%=5.1万 / 25%=6.4万 / 30%=7.7万
住宅返済"安全枠" = 返済負担率 20〜25%
既存債務（車ローン等）は安全枠から差し引いて"純枠"を出す
```

### 2-2. DTI（返済比率）の合格ライン
```
DTI = (住宅元利 + 他返済) ÷ 税込月収 ≤ 30〜33%
※ PCローン・車ローン等も必ず合算する
（返済5.5万・月収25.6万なら DTI≈21〜22% → 健全）
```

### 2-3. 総支出（=月の持ち出し）の年収帯別 勝ちレンジ
AIが統計的に「可決・維持が安定」と判定するゾーン。**ここに収めるのが鉄則。**

| 年収帯 | 総支出の安全レンジ |
|---|---|
| 300〜350万 | **6.0〜7.0万**（8万設定は余裕が薄くブレやすい） |
| 〜450万 単身 | **5.5〜6.0万**（家賃+αで意思決定が速い） |
| 450〜550万 | **6.0〜6.5万** |

> 総支出 ＝ 返済 ＋ 固都税 ＋ 火災地震保険 ＋ 修繕（区分は管理費・修繕積立金）

### 2-4. 借入概算の換算ルール
```
月返済 ±1万円 ≒ 借入 ±約330〜360万円
（前提：35年・金利1.0〜1.5%・ボーナス返済0）
```
→ これで顧客の前で即座に「持ち出し○万なら○千万円の物件」が暗算できる。

### 2-5. 固定費の標準係数（戸建）
```
固都税 0.8〜1.0万/月 ＋ 火災地震 0.1〜0.2万/月 ＋ 修繕 0.5〜1.0万/月
= 固定費 計 1.5〜2.5万/月
区分マンションは ＋管理費・修繕積立金 2.0〜3.5万/月（→総支出が上振れ）
```

### 2-6. 手元現金の死守ライン
```
実行後も 手元現金 ≥ 150万円（or 生活費6〜12ヶ月）を死守
→ 頭金の入れ過ぎNG。私募債など追加商材は現金確保を割るなら保留
```

### 2-7. GO / STOP 基準（合否しきい値）
```
GO   ：総支出 ≤ 上限 / DTI ≤ 33% / 事前照会◯ / 現金残 ≥ 生活費6〜12ヶ月
STOP ：上記を割ったら 価格・タイプ・立地・頭金を "即差替え"
```

> **使い方**: 商談中にこの7つの数字を頭で回せれば、その場で「この物件はGOかSTOPか」を即断できる。**迷わない営業＝決まる営業**。

## 3. AIが褒める「良い商談・良いアポ」の条件

高評価（【ひと言】で称賛）された案件に共通するポイント。**原文ママ**で引用します。

### 商談側
1. **エリア・条件の柔軟な拡大判断** — 「エリア拡大の判断が的確です。」
2. **正しい打ち切り（見切りの早さ）** — 「適切な見切り、ナイス判断です。」「見極めの早さが素晴らしいです。」
3. **初動で要点を丁寧に押さえる** — 「初動バッチリです。総支出を"家賃並み"で握り…」
4. **事実・数字を文面で固める** — 「丁寧な事実整理ができています。安心を数字で作って…」「信頼は数字で。」
5. **出口でも紹介導線を残す** — 「打ち切り時に"再開条件"と"紹介導線"を明文化した案件は、後日成約率と紹介発生率が高い。」

### アポ側
6. **次回アポ確定＋Wリマインド設計** — 「Wリマインド（中間SMS＋前日TEL＋当日OK返信）で来訪安定。」
7. **"数字1枚"で腹落ちさせる段取り** — 「数字で"道筋"を見せれば腹落ちします。準備8割。」
8. **顧客の目標金額・ニーズが明確** — 「目標金額が明確な方は『期間×期待利回り×月積立』の逆算提示で温度↑。」
9. **同行者（CMO/設計士）と当日アジェンダを事前手配** — 「設計士同席×事前回収で『可否を数値で即答』できれば勝ち筋が見えます。」

## 4. AIが繰り返し叩く「NG・改善点」

要改善でAIが必ず指摘する典型パターン。**ここを潰すだけで評価が一段上がる。**

### 4-1. 数字が "口頭" で止まっている（最頻NG）
- 「総支出の上限が未合意（返済額のみの会話リスク）。」
- 「総支出"7〜8万"の幅が大きい→7.0万（次点7.5万）で固定推奨。」
- 「年収は口頭→証憑未回収。」

### 4-2. 与信・書類が未回収
- 「与信未整備（CIC・事前照会）→可否と金利帯が不明。」
- 「書類未回収（源泉・給与3カ月・CIC・住民票）。」

### 4-3. 家計・収支が白紙 or 矛盾
- 「情報白紙（家計・貯金額・保険・DC制度）で設計精度が出にくい。」
- 「収入の矛盾（手取29.3万/月 vs 12万/月）→審査・資金計画が組めない。」

### 4-4. 手元資金が薄い／頭金入れ過ぎ
- 「手元資金90万円→実行後の流動性が薄い（最低100〜150万円を残す設計が必要）。」

### 4-5. 追加商材の押し過ぎ・順序ミス
- 「私募債等の同時提案は、住宅ローンの現金確保を下回ると失速。順序設計が鍵。」
- 「『大口NG』発言後は小口でも押すと信頼低下。」

### 4-6. メール未取得（証跡・到達性が弱い）
- 「メール未取得→証跡・到達性が弱い。」「連絡がTEL中心で…（メール取得必須）。」

### 4-7. 日程運用の不備・No-show リスク
- 「面談運用：中間確認の未設定／前日確認『17時』のみで日付不明。」
- 「次回まで約18日空き→温度低下とNo-showリスク。」「夜勤前合意のため当日離脱リスクあり。」

### 4-8. 区分マンションの管修上振れ
- 「区分MSは管修で総支出が上振れしやすい→同予算なら戸建の方が収まりやすい。」

### 4-9. 動機・時期が未定義 → 深追いNG
- 「目的・時期（いつ住み替えたいか）不明確。」「現時点で購入動機ゼロ→深追いは逆効果。」

## 5. AIが推奨する打ち手・トーク（コピペ可の武器）

### 5-1. 「総支出＝月の持ち出し」を文面で握る
> 文面合意：「月の持ち出し＝総支出」上限=7.0万（次点7.5万／ボ0）

### 5-2. 「総支出一枚表」で比較提示（最強の可視化武器）
> 候補3件（本命2＋比較1）を「総支出一枚表」（返済・固都税・火災地震・修繕/管修・DTI・必要頭金・実行後現金残）添付で提示

### 5-3. 物件なし事前照会 ×2行で金融の土台を作る
> 物件なし事前照会×2行（地元地銀/信金＋ネット）− 期間・金利帯・LTV・諸費用・団信を"文面"で取得

### 5-4. 追加商材は「順序」で切り返す（保留提案）
> 住宅ローンの事前◯＆物件GO確度>80%になるまで私募債入金は"保留"提案 − 代替：50万円に縮小 or 決済後に実行

### 5-5. アポの "数字1枚" 逆算トーク（無料・即決不要で温度を上げる）
> 「即決不要・費用なし・数字1枚」で（月いくら×何年×想定利回り）の逆算を可視化
> 例）目標4,000万円 → 30年×年率3%: 約6.8万/月 ／ 30年×年率5%: 約4.8万/月

### 5-6. Wリマインド運用（No-show撲滅の3段構え）
```
中間確認：前々日 SMS（書類催促）→必要時TEL
前日確認：前日17:00 TEL＋SMS（所要・持ち物再掲）
当日    ：当日12:00 最終SMS →"OK"返信必須（代替：±60分 or 別日）
```

### 5-7. 事前回収リスト（面談前に集める＝試算精度が跳ね上がる）
```
・源泉徴収票 or 直近給与明細1〜2ヶ月（年収確定）
・家計（月額）：家賃／食費／光熱／通信／交通／サブスク
・貯金残高（概算）／保険証券（有無・保険料）
・企業型DC：事業主掛金・マッチング可否・自己拠出額
・NISA：有無・積立額・商品／・車ローン：残債・金利・残期間
```

### 5-8. 打ち切り客への「再接触3択」＋「紹介依頼テキスト」（撤退を資産化）
> 再接触3択：A：◯カ月前に連絡OK ／ B：四半期の市況・金利配信のみ ／ C：連絡不要
> 紹介依頼（コピペ可）：「お知り合いで ①年収400万円以上 ②1年以内に購入検討 ③茨城・栃木・埼玉・都内or大阪圏 の方がいれば、氏名・携帯・希望エリアだけ私に送ってください。」

### 5-9. 団信NG懸念への切り返し
> 健康事由で団信が難しい案件は、フラット35（団信任意）やワイド団信の事前照会を"文面"で固める。

## 6. 結果ステージとAI分析の相関

AIは「結果」欄に応じて分析のトーンを切り替えています。

| 結果 | AI分析の主眼 | 【ひと言】のトーン |
|---|---|---|
| **再訪（前進中）** | 資金設計・物件方針・GO/STOPをフル展開／書類回収と"文面合意"を最優先アクションに | 「一気に前進」「この調子」と後押し |
| **打ち切り** | 資金設計は縮退、代わりに**出口対応の質**（再接触3択・紹介依頼・CRM休眠登録）を評価・指南。深追いを戒める | 「ナイス判断」「見極めの早さが素晴らしい」と撤退を肯定 |
| **生成中.....** | 分析前のプレースホルダ（中身なし） | — |

> **重要な示唆**: AIは交渉回数が多くても結果が出ない案件（例：13回交渉→打ち切り）に「タイミング不一致／休眠化が最適」と判定する。**回数を重ねること自体は評価されない。前進か、正しい撤退か**が問われる。

## 7. 勝ちパターン統合（AI横断の結論）

`sales-analysis` の全出力から抽出した、AIが認める勝ち筋：

1. **「総支出＝月の持ち出し」を最初に文面で握る**（年収帯別の安全レンジに収める）。
2. **数字を1枚にして見せる**（総支出一枚表／逆算1枚）。「信頼は数字で」。
3. **物件なし事前照会×2行を先に通す**。順序＝文面合意→書類→事前照会→内見→GO/STOP。
4. **手元現金を死守**（実行後≥150万 or 生活費6〜12ヶ月）。頭金入れ過ぎNG。追加商材は現金確保を優先し保留。
5. **区分MSより中古戸建**（同予算で収まりやすい）が基本路線。
6. **GO/STOPを数値で握り**、割ったら価格・築・立地・頭金を即差替え。
7. **見切りも勝ち筋**。動機ゼロは深追いせず、再接触3択＋紹介依頼＋休眠登録で出口を資産化。
8. **次回確定×Wリマインド×事前回収×メール取得**で歩留まりを設計。「準備8割」。
9. **同行者（CMO/設計士）と当日アジェンダ（時間配分付き）を事前手配**して主導権を握る。

## 8. 実装チェックリスト：AIに高評価される報告を書く

商談・アポ報告を投稿する前に、これを自己チェック。**AIの【課題】【未了】に先回りできれば一流。**

**数字（最優先）**
- [ ] 年収を**証憑**（源泉/給与明細）で確定したか（口頭はNG）
- [ ] 家計（家賃・固定費・食費・手残り）を実数で把握したか
- [ ] 貯金残高を概算で押さえたか
- [ ] 総支出の上限を**文面合意**したか（年収帯の安全レンジ内か）
- [ ] 既存債務（車・PC・カード）を合算してDTI≤33%を確認したか
- [ ] 実行後の手元現金 ≥150万円を確保できる設計か

**段取り**
- [ ] 次回日時を確定したか（長期空きを作っていないか）
- [ ] Wリマインド（中間/前日/当日OK返信）を設計したか
- [ ] メールアドレスを取得したか
- [ ] 事前回収リストを送ったか
- [ ] 同行者と当日アジェンダを手配したか

**判断**
- [ ] GO/STOPを数値で言えるか
- [ ] 動機ゼロ/属性NGなら、深追いせず出口（再接触3択＋紹介依頼）を作ったか
