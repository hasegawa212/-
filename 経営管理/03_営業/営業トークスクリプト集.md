# 現場トークスクリプト＆実用ツール集

> **使い方**: そのまま声に出して使える"逐語スクリプト"です。`[　]` は状況に合わせて差し替える変数。★は絶対に外さない要所。データ分析（『商談アポ個人別ランク別分析.md』等）で**実際に決まった案件に共通する言い回し**を型にしています。
> **大原則**: ①数字は口頭でなく"文面"で握る ②断られてから本番（前進は中央値6.5回目）③ランクで力の入れ方を変える（A＝全力／C・D＝低工数／E＝撤退）。

---

## 0. 全フェーズ早見（このスクリプトの地図）

| フェーズ | ゴール | 使うスクリプト |
|---|---|---|
| ① アポ獲得（飛び込み/反響） | 次回の日付を切る | §1 |
| ② 初訪 | 関係構築＋深いヒアリング＋次回「書類お預かり」の約束 | §2 |
| ③ 再訪（数字提示） | 総支出を文面で握る／物件を絞る | §3 |
| ④ 切り返し | 断り文句を次の論点に変える | §4 |
| ⑤ 三段ピボット | 不動産NGでも手ぶらで帰らない | §5 |
| ⑥ クロージング（決断取り） | その場で金銭/書類/契約日を動かす | §6 |
| ⑦ フォロー | バックれ・日時変更を潰す | §7 |
| ⑧ 撤退・紹介 | 失注を資産化 | §8 |

## 1. アポ獲得スクリプト

### 1-1. 飛び込み・インターホン（警戒MAXの相手）
> 「突然のご訪問で失礼します。[エリア名]で[将来のお金・税金]についてのご案内をしている[会社名]の[氏名]と申します。お売り込みではなく、**今やられている対策が今のままで大丈夫かの確認**だけ、3分だけお時間いただけませんか？」

断られたら（★引かずに信頼補完）:
> 「お忙しいところ失礼しました。怪しく感じられて当然です。**これ、うちの会社のホームページです**（スマホを見せる）。ご覧いただいて、もしお一人の時間が取れるタイミングがあれば、改めてご説明させてください。」

★ 日付を切る決め台詞（同業の"やりっぱなし"と差別化）:
> 「以前、似たような案内が来て連絡が来なかった、という方が本当に多いんです。**僕たちは"決めた日付に必ず来ます"。** 来週の[平日 or 土日]、[時間帯]はご在宅ですか？」

### 1-2. アンケート切り口（FP装い・不安喚起）
> 「[エリア]の方に**生活意識のアンケート**をお願いしています。年金・税金まわりで2〜3問だけ。──[2000万円問題]ってご存知ですか？」
> （知らない反応なら）「ですよね。実は今のままだと[老後／税金]で[数百万]損する方が多くて。**今の対策が合っているかだけ、一度数字でお見せします**。来週[日付]いかがですか？」

### 1-3. 反響・来場予約（bookrun/LP経由）
> 「この度はお問い合わせありがとうございます。[氏名]と申します。当日は**売り込みは一切なし**、[ご年収・今の家計]から『[ご希望]がいくらで叶うか』を**数字1枚**でお見せします。所要[45分]、[オンライン/ご自宅]どちらが楽ですか？」

### 1-4. 夜勤・多忙な相手（短時間で確定）
> 「お疲れのところすみません。今日は**来た目的だけ**お伝えして日付だけ頂きます。詳しいお話は前日に改めてご連絡しますので、[日付]の[時間]、5分だけ確保いただけますか？」

> 💡 **アポ獲得の鉄則**: ①必ず録音・画面録画 ②「会えるランク（A〜C±）」を即評価 ③その場で次回日時＋前日確認の段取りまで決める。

## 2. 初訪スクリプト（売り込まない・聞く）

初訪のゴールは契約ではなく **関係構築＋ヒアリング＋次回「源泉・身分証お預かり」の約束**。

### 2-1. アイスブレイク（褒めて感情を開く）
> 「お邪魔します。[お部屋／趣味の物]、いいですね。[具体的に一言褒める]。[出身]はどちらなんですか？」
> （会話を広げる：出身→仕事→休日→将来の不安、の順で自然に）

### 2-2. ★必ず聞く「動機・きっかけ」（感情の理由を作る）
> 「ちなみに、[マイホーム／資産形成]って、**どういうきっかけ**で考え始めたんですか？」
> 「もし叶うなら、**5年後どんな暮らし**をしていたいですか？」
> → スペックだけでは人は決断しない。"感情の理由"を必ず言語化させる。

### 2-3. 収支ヒアリング（後の数字提案の土台）
> 「失礼じゃない範囲で、お金の現状だけ整理させてください。今の**お家賃**は？／**月の手取り**はだいたい？／**固定費**（光熱・通信・サブスク）は？／**毎月いくら残り**ますか？／**貯金**はざっくり？／車や[カード]のローンは？」

### 2-4. 次回アポ（書類お預かりの約束＝本気度の証）
> 「次回は、[ご年収]で実際にいくらいけるかを**数字でお見せ**します。それまでに**源泉徴収票（or 給与明細1〜2ヶ月）と身分証**だけご準備いただけますか？ 写真でも大丈夫です。[日付]いかがですか？」

## 3. 再訪スクリプト（数字を見せて握る）

### 3-1. 総支出を"家賃並み"で文面合意（★最重要）
> 「前回伺った[ご年収]だと、**無理のない月の持ち出しは[5.5〜6.0]万円**が安全圏です。今の家賃[8]万円より[安く]、しかも資産になります。
> ──この『**月の持ち出し＝総支出[6.0]万円まで**』、ここを今日握らせてください。これ以上は絶対に超えないお約束です。」
> （※年収帯の安全レンジ：300-350万→6.0-7.0万／〜450万単身→5.5-6.0万／450-550万→6.0-6.5万）

### 3-2. 「総支出一枚表」で見せる（信頼は数字で）
> 「ふわっとした返済額だけだと不安ですよね。だから**1枚にしました**。──返済・固定資産税・火災保険・修繕費、ぜんぶ込みで『月いくら』『手元にいくら残るか』。区分マンションは管理費・修繕積立金が乗るので、同じ予算なら**中古戸建のほうが収まりやすい**んです。」
> （→ §ツール「総支出一枚表テンプレ」を提示）

### 3-3. 借入の暗算で即答（できる営業に見せる）
> 「月の返済を1万円増やす／減らすと、借りられる額が**約330〜360万円**動きます。なので[6万]返済なら、ざっくり[◯◯]万円までの物件が射程です。」

### 3-4. 物件提案（中間価格の本命＋比較で即決を誘う）
> 「2軒お持ちしました。**価格も立地も"中間"のこちらが本命**、もう1軒は比較用です。次回[内見]で実物を見て、その場で[どちらでいくか]決めましょう。」

## 4. 切り返しスクリプト（断り文句別・逐語）

### 4-1. 嫁ブロック・家族反対
相手:「家族（妻/親）に反対されていて…」
> 「ご家族が心配されるのは**当然**ですし、むしろ良いご家族ですね。であれば、**次回はご一緒に**お話しさせてください。借入を増やすのでなく『家計がむしろ安全になる』のを、**ご家族にも数字で**お見せします。本人だけで進めると、後でひっくり返って一番もったいないので。」

### 4-2. ローン不安・独身不安
相手:「独身でローンなんて周りにいないし不安で…」
> 「その不安、めちゃくちゃ普通です。実は[同じ20代・独身・同じ職種]の方、[たくさん]いらっしゃいます（事例を見せる）。それに今の家賃[8万]は**掛け捨て**ですが、これは**払った分が資産**になります。持ち出しも月[6万]に抑えるので、生活は今と変わりません。」

### 4-3. 転勤・地元回帰
相手:「[2年後]に[地元/都内]に戻るかもなので…」
> 「なるほど、それなら**住む前提のマイホームは一旦置きましょう**。代わりに『住まなくても資産になる』形（投資用 or [私募債]）が合います。戻られても困らない設計に切り替えてご説明しますね。」

### 4-4. 他社検討・商品比較
相手:「他社も回っていて／投資信託で同じ利回り出せるし…」
> 「しっかり比較されていて素晴らしいです。最後は**"誰から買うか"**だと思っています。僕は[決めた日付に必ず来る・レスが速い・全部数字で出す]。利回りだけでなく、**流動性・確実性・手間**まで含めて比べると見え方が変わります。そこも1枚で出しますね。」

### 4-5. 温度感ゼロ・取り組む気なし（→見切り）
相手:「正直やる気はないです／一度断ったので」
> 「承知しました。無理にお勧めはしません。今日はここまでにします。」
> → ★ **深追いしない**。打ち切り判断（Eランク）。§8の出口・紹介へ。

### 4-6. 属性NG（貯金ゼロ・延滞）
> （ローンが通らない＝商談不成立）アポ段階で見極める。発覚したら無理に進めず、§5の保険/リフォーム、または丁寧に終了。

### 4-7. 団信が不安（健康事由）
> 「健康面が気になるなら、**団信が任意のフラット35**や**ワイド団信**という選択肢があります。通るかどうか、先に銀行へ"文面"で確認しておきますね。」

## 5. 三段ピボット・スクリプト（手ぶらで帰らない）

```
① 不動産（マイホーム/投資用）  → ダメなら
② 私募債（1口100万円）        → ダメなら
③ 保険 / リフォーム
```

不動産→私募債:
> 「不動産が今じゃない、よく分かりました。であれば、まとまった資金を[年率◯%]で運用できる**私募債（1口100万円）**はいかがですか。住まいの決断は要りません。」

私募債→保険/リフォーム:
> 「[ご貯金]だと1口100万は手元が薄くなりますね。それなら無理せず、まずは**[保険の見直し]／[今のお住まいのリフォーム]**で[月◯万]の改善から始めましょう。」

> 💡 断られたのは"商品"であって"あなた"ではない。必ず次のカードを切る。

## 6. クロージング・スクリプト（決断取り）

### 6-1. 決断取り（逃げ場を作らず次工程を先に確定）
> 「2物件とも気に入っていただけましたね。では**次回、申込書を書きましょう**。それまでに**どちらでいくか**だけ決めておいてください。日付は[◯/◯ 14時]、ここで[契約]、続けて[15時に本申込]まで進めます。」
> ★ ポイント:「どちらにしますか？」と聞かない。**「次回は書類を書きます」と工程を先に固定**してから選択を迫る。

### 6-2. 質疑応答を全部返してから握る
> 「ここまでで、**気になる点・不安は全部出し切りましたか？**（全部返答する）── では、[マイホーム]、僕に任せていただけますか。」

### 6-3. 背中を押す（CB＝キャッシュバック）
> 「ご決断いただけるなら、[家賃半年分／3万円]を**キャッシュバック**します。今損したくないお気持ちに、**確実な得**でお応えします。」

### 6-4. 金銭・書類・日付を"その場で動かす"
> 「では**手付金[50万円]**は[契約日]までにご準備ください。今日は**免許証・保険証のお写真**と**ヒアリングシート**をお預かりします。」
> ★ 決まった商談は必ず「金銭が動く／書類を預かる／次回契約日が決まる」のどれかが起きる。「前向きに検討」で終わったら未達。

### 6-5. 上席同行の使い方（自分で土台→決断だけ借りる）
> （事前に報告テンプレで顧客情報を共有した上で）商談冒頭に上席へ電話を繋ぎ、本人の口から次回確約を取る＝「電話グリップ」。決断取りの一押しを上席に任せる。

## 7. フォロー・スクリプト（Wリマインドでバックれ撲滅）

慢性的な「日時変更（月20〜31件）」「バックれ」を仕組みで潰す。

**中間確認（前々日・SMS）**
> 「[お名前]様、[◯/◯ ◯時]のお約束の確認です。ご準備の[源泉・身分証]、当日写真でも大丈夫です。何かあれば前日に調整しますね。」

**前日確認（前日17時・TEL＋SMS）★を付ける**
> 「明日[◯時]、[所要45分]でお伺いします。[持ち物]だけ念のため。お気をつけて。」
> → 取れたら報告に「★」（前日確認OKの印）。

**当日（12時・最終SMS／"OK"返信必須）**
> 「本日[◯時]、よろしくお願いします。問題なければ"OK"だけご返信ください。難しければ[±60分 or 別日]で調整します。」

> 💡 メールは必ず取得（TELだけは証跡・到達性が弱い）。

## 8. 撤退・紹介スクリプト（失注を資産化）

### 8-1. 再接触の許諾（3択で関係を残す）
> 「今日は無理にお勧めしません。今後の連絡、3つから選んでください。**A：[戻られる◯ヶ月前]に一度ご連絡 ／ B：金利・市況の情報を四半期に1回だけ ／ C：連絡不要**。どれが良いですか？」

### 8-2. 紹介依頼（コピペで送れる定番）
> 「もしお知り合いで、**①年収400万円以上 ②1年以内に購入を検討 ③[茨城・栃木・埼玉・都内]** の方がいれば、**お名前・携帯・希望エリアだけ**僕に送ってください。必ず丁寧に対応します。」

> 💡 打ち切り時に「再開条件」と「紹介導線」を明文化した案件ほど、後日の成約率・紹介率が高い（データ実証）。

# 実用ツール

## ツール1：事前回収リスト（再訪前に送る）
```
次回までに、写真 or 箇条書きでご準備ください（試算の精度が上がります）
□ 源泉徴収票 または 直近給与明細1〜2ヶ月
□ 家計（月額）：家賃／食費／光熱／通信／交通／サブスク
□ 貯金残高（概算）
□ 保険証券（有無・月額）
□ 企業型DC：会社掛金・マッチング可否・自己拠出
□ NISA：有無・積立額・商品
□ 車/カードローン：残債・金利・残期間
```

## ツール2：総支出一枚表（提案テンプレ）
| 項目 | 物件A（本命） | 物件B（比較） |
|---|---|---|
| 物件価格 |  |  |
| 月返済（35年・金利◯%） |  |  |
| 固定資産税（/月換算） |  |  |
| 火災・地震保険（/月） |  |  |
| 修繕（戸建）/ 管理・修繕積立（区分） |  |  |
| **総支出（月の持ち出し）** | **★** | **★** |
| DTI（返済比率） | ％ | ％ |
| 必要頭金 |  |  |
| 実行後の手元現金 | ≥150万 | ≥150万 |

> 合格ライン：総支出≤年収帯の安全レンジ／DTI≤33%／手元現金≥150万円。割れたら価格・タイプ・立地・頭金を即差替え（STOP）。

## ツール3：商談前の30秒セルフチェック
- [ ] この客の**ランク**は？（A=全力／B=育成／C・D=低工数／E=撤退）
- [ ] 今日のゴールは「金銭/書類/次回契約日」のどれを動かす？
- [ ] **総支出の上限**を文面で握れているか？
- [ ] 次回日時＋前日確認の段取りは決めたか？
- [ ] 録音・画面録画はONか？

## ツール4：報告テンプレ（商談後すぐ投稿）
```
【面談日】／【結果】再訪◯/事前申込/契約本申込/私募債/保険/リフォーム/打ち切り
■基本情報：氏名/年齢/勤務先/勤続/年収
■商談概要：商談ランク/交渉回数/結果
■担当・同行：担当/同行者
■内容要約：何が起きて、次に何をするか（★ここが命）
■次回予定：訪問日/契約日/決済日
■預かり書類：免許→保険証→ヒアリングシート→CIC→源泉→…
```

## ツール5：年収帯別の安全レンジ早見
| 年収帯 | 総支出（月の持ち出し）の安全圏 |
|---|---|
| 300〜350万 | 6.0〜7.0万 |
| 〜450万（単身） | 5.5〜6.0万 |
| 450〜550万 | 6.0〜6.5万 |
| 借入暗算 | 月返済±1万 ≒ 借入±330〜360万（35年・金利1.0-1.5%・ボ0） |
