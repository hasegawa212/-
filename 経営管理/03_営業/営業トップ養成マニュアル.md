## 0. 結論ファースト：トップ営業がやっている3つのこと

全データを分析して見えた、**決める人と決めない人の差**はこの3点に集約されます。

1. **「次の一手」を商談の場で必ず確定させる** — 決まる商談は、その場で必ず「金銭が動く・書類を預かる・次回契約日が決まる」のどれかが起きている。「前向きに検討」で終わる商談は流れる。
2. **断られてから本番だと知っている** — 商談結果の最頻値は「再訪」。1回で決まらないのが前提。**断り文句を"終了"ではなく"次の論点"に変換**できる人が勝つ。
3. **上席を"武器"として設計して使う** — 事前申込・契約・私募債・打ち切りの判断はほぼ全て上席同行回で発生。**いつ・誰を・何のために同行させるか**を逆算できる人がトップになる。

この3点を、アポ→商談→クロージングの流れに沿って分解していきます。

## 1. 全体像：案件はこのファネルを通る

```
リード獲得 → アポ獲得 → 初回訪問(初訪) → 再訪(複数回) → 物件提案 → 内見 → 決断取り
   → 事前審査(事前申込) → 契約 → 本申込 → 金消契約 → 決済 → 歩合確定
```

**各ステージの用語（社内語彙・正しく使えること自体がプロの証）**

| 用語 | 意味 |
|---|---|
| 初訪 | 初回訪問。関係構築とヒアリングが目的。ここで売り込まない |
| 再訪①②… | 2回目以降の訪問。番号で何回目かを管理。**大半の案件は再訪3〜7回で決まる** |
| 物件提案 / 物紹 / 振替 | 物件を提案するフェーズ。刺さらなければ"出し直し（振替）" |
| 内見 | 物件を実際に見せる。**即決交渉の勝負どころ** |
| 決断取り | 複数物件を1つに絞らせ、意思決定させる工程。**契約直前の最重要クロージング** |
| 事前審査 / 事前申込 | 住宅ローンの事前審査に申込。銀行名が入る＝有力前進のサイン |
| 本申込 | ローン本申込。契約と同日に連結させるのが理想 |
| 金消契約（きんしょう） | 金銭消費貸借契約。ローン実行の契約 |
| 決済 | 物件引渡し・代金決済。**ゴール。ここで歩合が確定** |
| 手付金 | 契約時に顧客から受領する金銭。**振込ではなく現金で武田CMOに一元管理**（社内ルール） |
| 打ち切り | 失注確定。見込みなしと判断して終了。判断はほぼ上席 |

> **新人へ**: この用語を「正確に」「正しいタイミングで」使えるかが、顧客と上席からの信頼の第一歩。報告で用語を間違えると案件管理が崩れる。

## 2. アポ獲得：質の高いアポだけが商談を生む

### 2-1. アポの獲得経路（全6ルート）

| 経路 | 内容 | 特徴 |
|---|---|---|
| **訪問営業（飛び込み）** | 最多。ピンポンで対面化 | 警戒心が最大の壁。室内観察まで報告 |
| **アンケート起点（FP装い）** | 「生活意識調査アンケート」を切り口に対面化 | 税金・年金・2000万円問題で不安喚起 |
| **反響アポ（LP）** | Web広告/LPからの問い合わせ | 担当=「反響LP」。オンライン初訪が多い |
| **来場予約（bookrun）** | サイト予約フォーム→Zoom商談 | 流入元 `bookrun`。確度が比較的高い |
| **紹介・保険アポ** | 既存客や保険からの横展開 | クロスセルの起点 |
| **追客→復活アポ** | 一度離れた客を再訪で復活 | 「直接伺って新しく日時を頂いた」 |

### 2-2. ターゲット像（誰にアポを取るべきか）

実データの成約・有力案件に共通する顧客プロファイル＝**狙うべき客**:

- **年齢**: 25〜40歳（若手中心）
- **職業**: 製造業・半導体・自動車製造・派遣技術職（メイテック/テクノプロ/UT等）の**正社員**
- **年収**: 350〜600万（マイホーム）／高年収・高貯金層には私募債・投資用不動産
- **与信**: **借入なし・扶養なし・延滞なし**（投資用ローンが通る属性を選別）
- **投資状況**: NISA/iDeCoを「やってるが中途半端」「考えてるがやれてない」層が最も刺さる
- **エリア**: 北関東の賃貸（レオパレス/アパート）単身者

> **逆に避けるべき（打ち切り直結の属性）**: 年収270万台・クレジット延滞・CIC事故情報・貯金0円。これらは商談に乗せても上席判断で打ち切りになる。**アポ段階で見極める**のがトップの効率。

### 2-3. アポ報告テンプレート（この型で報告する）

```
見込み報告（Sales Analysis）
担当：（氏名）
■ お客様情報
・氏名／年齢／住所／職業（＋勤続年数）／年収／貯金／最終学歴／借入／扶養／出身
■ アポイント情報
・アポ日／会えるランク（A〜C±）／次回交渉日時／電話確認担当／中間確認／前日確認
■ 連絡先（電話／メール／SNS）
■ 商談メモ
・落とし内容（トークの切り口）／ボイスレコーダー有無
■ 性格・印象（希望同行者を含む）
■ プライベート情報（配偶者／結婚願望）
■ 備考（観察メモ・刺さったポイント・注意点）
```

> **なぜここまで書くのか**: 同行する上席が**会う前に顧客を理解する**ため。備考の「褒めるとめっちゃ笑顔」「会社紹介をしっかりした方が良い」といった一言が、次回商談の作戦になる。**報告の質＝チーム戦の質**。

### 2-4. 会えるランク（A/B/C ±）の運用

確度を **A > B > C**、微調整を **+ / −** で評価。実データの新規アポは **C〜C+ が中心**＝「最初から確度の高い客などほぼいない」。だからこそ次の確認フローで確度を引き上げる。

### 2-5. アポを「流さない」多段確認フロー（標準装備）

```
アポ取得 → 中間確認（SMS/TEL）→ 前日確認（★を付ける）→ 商談入室
```

- **★付け＝前日確認が取れた印**。前日確認が取れないアポは飛ぶ（実例: 前日確認が取れず当日訪問→不在=バックれ）。
- **アポインターと確認担当を分業**（「○○代理」が電話確認を担当）。属人化させず仕組みで詰める。

### 2-6. アポ獲得トーク 鉄板集（実データから）

警戒心の強い飛び込み客から日付を取るための、実際に効いた言い回し:

1. **「決めた日付に必ず来ます」と言い切る**（最重要）
   > 過去にFPを名乗る人間が来てアンケートだけ取って音沙汰なし → 警戒MAXの客に対し「**僕たちは決めた日付に必ず来る**」と伝えて日付獲得。
   → 同業の"やりっぱなし"と自分を差別化する一言。

2. **断られてもホームページを見せて信頼補完、再訪余地を残す**
   > 「一回断り入った／訪問多いと警戒あり → **ホームページを見せた**。また見てほしいと伝えた」

3. **「無理にでも時間を設けてもらっている」と前提化して日付を切る**
   > 「ちゃんとやっていると言い張る客に、**"無理くり時間を設けていただいている"とお伝えして日付を切った**」

4. **褒めて感情を開かせる** … 「褒めるとめっちゃ笑顔で喜ぶ」タイプは肯定から入る。

5. **夜勤客は"目的だけ"伝えて即アポ→詳細は前日に**
   > 「夜勤前の対応で、**来た目的だけ説明してお日付頂いた**。詳細は前日に改めて説明」

6. **不安喚起の切り口（落とし内容）の定番**: 「税金対策」「2000万円問題」「今のNISA含め見直し」「FPの存在意義」「将来対策」。**"知らなかった"を起点に不安を作る**。

7. **録音・画面録画を必ず取る**（社内ルール）。「初面談録音と画面録画わすれずに！」。教育と再現性の生命線。

## 3. 商談：初訪からクロージングまで

### 3-1. 商談結果の分類（あなたのゴール設定）

商談は必ず以下の「結果」で締める。**最頻値は「再訪」＝1回で決めようとしない**。

| 結果 | 状態 | 次の狙い |
|---|---|---|
| 再訪①②… | 継続。関係構築・ヒアリング段階 | 番号を進める。停滞させない |
| 再訪(物件紹介/振替) | 提案フェーズ | 刺さらなければ物件を出し直す |
| 再訪(決断取り) | クロージング直前 | 1物件に絞らせる |
| 事前申込 | 銀行の事前審査へ | 契約日を即確定 |
| 契約本申込 | 最終段階 | 契約と本申込を同日連結 |
| 私募債契約 | 不動産NG客の受け皿 | 現金1口100万円を預かる |
| リフォーム申込 | 手付金受領 | 申込書記入まで完了 |
| 保険切替 | 不動産・私募債も不可 | 最後のピボット先 |
| 媒介契約締結 | 売却仲介（最高難度S） | 上席のカリスマ案件 |
| 打ち切り | 失注 | 上席判断。深追いしない |

### 3-2. 商談報告テンプレート（この型で記録する）

```
【面談日】／【チャンネル】#03_商談報告
■ 基本情報    顧客名/年齢/住所/勤務先(出向・派遣先)/勤続/年収/ボーナス
■ 商談概要    名刺/相談シート/商談ランク/交渉回数/結果
■ 担当／同行  経路/担当営業/同行者①②/同行回数/担当業務
■ 訪問記録    IN/OUT/交渉場所/ボイレコ
■ 内容要約    ← 商談の本文。最重要。何が起きて次に何をするか
■ 物件・金融情報 物件名/金融機関/自決日/借入/加入保険/貯金・手付金
■ 次回予定    訪問日/契約日/決済日
⚠必ず聞く⚠   購入の動機・きっかけ／趣味から購入イメージを広げる
■ 備考＋ヒアリング 出身/休日/趣味/将来対策/車/結婚願望
              ＋ 月手取り/家賃/固定費/食費/趣味/手残り/対策に回せる資金
■ 預かり書類【取得順】免許証→保険証→ヒアリングシート→CIC→公的→源泉(1〜3期)
              →事前申込書→本申込書→返済表→同意書→承諾書→マイホム→個人情報同意書
```

> **`⚠必ず聞く⚠`の意味**: 「購入の動機・きっかけ」「趣味から購入イメージを広げる」は、**ローンを組む決断を支える"感情の理由"**を作るための必須ヒアリング。スペックだけでは人は決断しない。

### 3-3. 初訪の鉄則：売り込まない

実データの初訪報告に共通するのは「**不動産の話まで"させていただいた"／前向きに考えていただく**」という到達点。初訪のゴールは契約ではなく:

1. **関係構築**（警戒を解く）
2. **深いヒアリング**（収支・家族・将来・動機）
3. **次回アポの確定**（再訪②で源泉・身分証を預かる約束まで）

> 初訪で「源泉・身分証を次回お預かりする」約束まで取れたら成功。**書類を預ける＝顧客の本気度のバロメーター**。

### 3-4. 物件提案と内見：即決を生む物件選定術

部長コメントより、内見で即決させる物件の出し方:

> **「価格も距離も中間ぐらいの物件が出ると内見日に即決交渉しやすい。もう一軒の物件選定をして内見の段取りを組む」**

- **2軒以上を用意し、価格・立地が"中間"の物件を本命に置く** → 比較対象があることで本命が際立ち、その場で決断させやすい。
- 内見は「見せる」ではなく「**即決交渉の場**」と設計する。

### 3-5. 決断取り：契約直前の最重要クロージング

**「決断取り」＝複数物件を1つに絞らせ、意思決定させる工程**。これができないと永遠に「検討」で止まる。

実例の型:
> 「2物件ともいいが金額で一度考えたいとのこと → **次回、書類記入をお伝えした上で、どの物件でいくか決断取りさせていただく**」

ポイント: 「どちらにしますか」ではなく「**次回は書類を書きます。それまでにどちらか決めてください**」と、**次工程（書類記入）を先に確定**させてから選択を迫る。逃げ場を作らない。

### 3-6. クロージングが成立した瞬間の共通言語

決まった商談には必ずこの表現が出る。あなたの商談でこれが言えたか自己採点せよ:

- **「無事ご納得していただきました」** — 納得が言語化されている
- **「決断取りに入らせて頂き、質疑応答を全て返答した上で、任せて頂くことになりました」** — 反論を残さない
- **その場で金銭が動く**: 「手付金50万円を受け取らせて頂きました」「現金で1口100万円をお預かり」
- **次回契約日が確定**: 「6/7（日）14:00契約 15:00本申込」「当日までに手付金をご準備いただく」

> **逆に流れる商談の言葉**: 「前向きに考えていただきます」「一度考えたい」だけで金銭も書類も日付も動いていない。

## 4. 切り返し大全：断り文句を"次の論点"に変える

商談結果の最頻値が「再訪」である以上、**切り返しこそが営業力の本体**。実データの断り文句を全カテゴリ分類し、対処法を示す。

### 4-1. 嫁ブロック / 家族反対
**実際の声**:
> 「家族から、借入が多いのに新しくローンを組むことへの懸念」「親に相談しないと進められない」「弟が不動産会社勤務で反対された」

**対処**:
- 反対している家族を**敵にしない**。「ご家族が心配されるのは当然」と受ける。
- **キーパーソン（決裁者）を商談に同席させる**設計に切り替える。本人だけ説得しても家でひっくり返る。
- 家族の懸念（借入・管理）に対し、**収支シミュレーションと持ち出し額の明示**で「むしろ家計が安全になる」を見せる。
- それでも固いなら**不動産→私募債/保険へピボット**（後述）。

### 4-2. ローン不安 / 独身不安
**実際の声**:
> 「独身でマイホームを買う身近な事例がない」「持ち出しを払い続ける不安」「親からまだ早いと言われた」

**対処**:
- **身近な事例の不在**が不安の正体 → **同属性（独身・同年代・同職種）の事例**を見せて「普通のこと」に変える。
- 「持ち出し」を**家賃との比較**に置き換える（今払っている家賃は掛け捨て、という再フレーム）。
- 月の手残り・対策に回せる資金をヒアリング済みなら、**無理のない持ち出し額（月6万円以内が標準）**を提示。

### 4-3. 転勤 / 地元回帰
**実際の声**:
> 「20代のうちに都内支社へ戻りたい」「2年以内に地元へ戻る予定」「彼女と埼玉で同棲予定」

**対処**:
- 居住前提が崩れるので**マイホームは無理筋**。早期に見切る。
- **投資用不動産（賃貸に回せる）or 私募債/保険へ即ピボット**。「住まなくても資産になる」へ切り替える。

### 4-4. 他社検討 / 商品比較
**実際の声**:
> 「不動産会社と工務店を3社回って話を聞いている」「投資信託で同等利回り（4.3%）が狙える、3年資金拘束にメリットを感じない」

**対処**:
- 比較されている時点で**温度はある**。価格勝負ではなく**"誰から買うか"の信頼**で差別化（決めた日付に必ず来る／レスの速さ／仕組み化された対応）。
- 商品スペック比較には**流動性・確実性・手間**など利回り以外の軸を足す。

### 4-5. 取り組み拒否 / 温度感ゼロ（→打ち切り判断）
**実際の声**:
> 「知識がないものを言うことを聞いてやることは絶対にない」「一度断っているからそもそも取り組むつもりがない」「断るを前提で内見に来ている」

**対処**:
- **これは追わない**。温度感ゼロは時間の浪費。上席判断で**打ち切り**にして、リソースを有力客へ。
- トップ営業の効率は「**追う客と捨てる客の線引き**」で決まる。

### 4-6. 属性NG（年収・延滞・貯金ゼロ）
**実際の声**:
> 「年収270万・クレジット延滞・1年以内に遅延複数 → 打ち切り」「貯金0円・残債・CIC事故情報 → 武田CMO判断で打ち切り」

**対処**: ローンが通らない＝商談が成立しない。**アポ段階で見極め**、商談に乗せない。

## 5. 三段ピボット：不動産がダメでも手ぶらで帰らない

トップ営業は**1つの商材に固執しない**。実データの黄金パターン:

```
① 不動産（マイホーム/投資用）
      ↓ ダメなら
② 私募債（1口100万円）  ← 高貯金・投資用所有層の受け皿
      ↓ ダメなら
③ 保険 / リフォーム      ← 最後のピボット先
```

**実例**:
- 「不動産は考えた結果やらない → **私募債に切り替え提案**」（張り込みで捕獲後）
- 「弟に不動産を反対された → **私募債のご提案に切り替え**」
- 「埼玉へ同棲で引っ越す → **次回保険に切り替えてお話し**」
- 「私募債200万償還要望 → **リフォーム150万円に転換**」

> **思想**: 顧客の状況（家族・転勤・属性）で最適な商材は変わる。**断られたのは"不動産"であって"あなた"ではない**。手元のカードを切り替えて必ず何かを残す。

## 6. 上席同行：最強の武器を設計して使う

### 6-1. 事実：決定打はほぼ上席同行回で出る

事前申込・契約本申込・私募債契約・打ち切り判断の**ほぼ全てが武田CMO/三浦部長/上田課長の同行回**で発生している。**クロージングと見切りは上席の役割**として設計されている。

> **⚠ 数字での補足（『商談アポ定量分析レポート.md』より）**: 直近の商談22件中**21件(95%)が上席同行**でした。つまり上席同行は"例外"ではなく**標準運用＝前提条件**。だから「呼ぶか否か」では差がつきません。**差がつくのは、同行を活かして商談の中で"数字を文面で握り・決断取りに持ち込み・正しく見切る"こと**。以下は"何のために呼ぶか"の設計図として読んでください。

### 6-2. 同行クローザーの役割分担（誰を何のために呼ぶか）

| 上席 | 強み | 呼ぶべき場面 |
|---|---|---|
| **武田CMO** | 最強クローザー兼最終判断 | 事前申込・契約・私募債のクロージング、打ち切り判断 |
| **三浦部長** | 関係構築・物件/土地振替・内見納得 | 前進（再訪・内見・物件納得）を量産したい時 |
| **上田課長** | リフォーム申込・私募債・手付回収 | 手付金を伴う申込を締めたい時 |
| **代表（長谷川光）** | カリスマ・最高難度交渉 | 媒介締結など S級・含み損確定の決断取り |

### 6-3. 上席を使ったクロージング戦術（実データ）

1. **電話グリップ**: 商談の冒頭で武田CMOに電話を繋ぎ、**次回訪問を本人の口から確約**させる張り込み戦術。
   > 「話の初めに武田CMOにお電話を繋げて頂き、次回伺うことをお伝えして頂いた」

2. **張り込み捕獲**: 不在・逃げ腰の客を物理的に捕まえる。
   > 「武田CMOに張り込みで捕まえて頂き、不動産はやらないとのことで私募債に切り替え提案」

3. **カリスマ決断取り（S級）**: 含み損を確定させてでも売却を決断させる。
   > 「10年前の信頼関係を入口に、含み損を確定させてでも売却を決断させたのは代表の圧倒的な自信と交渉力。**部長の関係構築 × 代表のカリスマ**で成立」

### 6-4. 同行を活かす新人の動き方

- **同行は"丸投げ"ではない**。あなたが関係構築・ヒアリング・物件提案で土台を作り、**最後の決断取りだけ上席の力を借りる**。
- 同行前に**アポ/商談報告で顧客情報を完全共有**（だからテンプレートを丁寧に埋める）。
- 同行回数も記録（「同行回数7回」）。**何回同行して決まったか**が自分の伸びしろの可視化になる。

## 7. 背中を押す仕掛け：CB（キャッシュバック）

最後の一歩は**経済的インセンティブ**で押す。

- 「物件購入したら**3万円プレゼント**すると伝えた」
- 「**家賃半年分（30万円）CB**」

> 迷っている客の「損したくない」心理に、**確定した得（CB）**をぶつける。ただし乱発せず、決断取りの最後の一押しとして使う。

## 8. KPI と数字管理：トップは自分の数字を即答できる

### 8-1. 管理される指標（ファネル順）

- **アポ本数**（月間目標に対する進捗。例: 個人目標 月4〜6本）
- 内訳: **メタ広告アポ / 紹介アポ / 保険アポ**
- **見込み**（アポ未満の手応え案件）
- **A/Y/K 集計**（個人別 アポ/約束/契約の3指標）
- **契約本数**（不動産＋保険を別建て）
- **決済**（決済金額・歩合対象か）
- **進行中案件**（自社案件／他社案件で決済予定をパイプライン管理）

### 8-2. 目標 vs 実績（実データの読み方）

実データ（6月）では月間アポ目標に対し上旬の実績が極端に低調（例: 全体 0〜2本/24本）。**新規アポ生成がボトルネック**。一方、進行中案件（7件前後）は安定し、約2,900万円の決済が着地。

> **学び**: パイプライン（進行案件）が安定していても、**新規アポという入口が枯れると数ヶ月後に必ず詰まる**。トップは「決済を上げながら同時に新規アポを切らさない」二正面作戦をやる。

### 8-3. 標準数値（提案の相場観）

- **持ち出し希望**: 月6万円以内が標準ボリュームゾーン（リフォーム系は月3万円以内）
- **私募債**: 1口100万円
- **手付金**: 50万円前後（契約時に現金受領→武田CMOへ手渡し一元管理）
- **狙う年収帯**: マイホーム350〜600万、私募債・投資用は高年収・高貯金層

## 9. 報告の哲学：動く"前"に共有しろ

マネージャー号令（実データ）より、**報告は記録ではなく戦略**:

> 「内見の予約が入ったら、内見の"前"に必ずフォームを送る。**動いてからの報告はただの記録。動く"前"の共有は戦略だ。**」
> 「連絡の手間ゼロ、伝達漏れゼロ。**気合いじゃなく仕組みで詰める。**予約を取ったら、まずフォーム。」

- **アポ/商談の前にフォーム共有** → 上席が会う前に作戦を立てられる＝チーム戦の質が上がる。
- **ボイスレコーダー・画面録画を必ず取る** → 振り返りと教育の生命線。
- 報告テンプレートを丁寧に埋めること自体が、**同行者という最強リソースを動かすための投資**。

## 12. トップ営業 自己採点チェックリスト

商談のたびに、これで自分を採点せよ。**「はい」が多いほどトップに近い。**

**アポ**
- [ ] 「決めた日付に必ず来る」と言い切ったか
- [ ] 会えるランクと次回交渉日時まで確定したか
- [ ] 前日確認（★）が取れているか
- [ ] 録音・画面録画を取ったか

**初訪・ヒアリング**
- [ ] 売り込まず、関係構築に徹したか
- [ ] 購入の「動機・きっかけ（感情の理由）」を聞けたか
- [ ] 収支（手取り/家賃/固定費/手残り/対策資金）を把握したか
- [ ] 次回「源泉・身分証お預かり」の約束を取れたか

**提案・クロージング**
- [ ] 中間価格の本命物件を含む2軒以上を用意したか

**チーム戦**
- [ ] 動く"前"にフォーム共有したか
- [ ] 上席を「いつ・誰を・何のために」設計して同行させたか
- [ ] 温度感ゼロ・属性NGを早期に見切ったか
