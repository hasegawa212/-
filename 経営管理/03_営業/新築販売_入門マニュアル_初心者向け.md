# 新築販売 入門マニュアル（初心者の再現性版 v1）

**このマニュアルのゴール** 入社して間もない人が、**先輩に聞かずに、東亞地所モデルの新築販売案件を1人で前に進められる**状態になること。専門用語・銀行手続き・お客様への伝え方まで、初めての人を前提にゼロから説明する。迷ったら口頭で聞く前に、まずここを探す。

> 体系版（全パターン網羅）は `東亞地所モデル_新築販売戦略マニュアル.md` を参照。**こちらは「初めての人が手順通りに動けばゴールまで行ける」入門版**。

## 0. まず全体像（私たちは何を売っているのか）

私たちは「土地＋注文住宅（建物）」をセットで、**サラリーマン世帯のお客様が住宅ローンを組んで買える形**にして売っている。判断の軸はいつもこの2つ。

1. **お客様が住宅ローンを組めるか**（年収・勤続・年齢に対して銀行が貸すか）
2. **月々の返済が、お客様の家計で無理なく払えるか**（目安：月8〜10万円）

この2つから逆算して、物件総額（4,000〜4,600万）・借入期間（40〜50年）・銀行を選ぶ。だから営業は「家を売る人」であると同時に**「お客様のローンを通す人」**でもある、と覚えておく。

### 登場人物（誰が何をするか）

| 主体 | 役割 | 主なメンバー |
| :-- | :-- | :-- |
| 東亞地所（仲介） | 物件・銀行手配、お客様対応、現場進行 | 石田氏・盛江氏・SUZUKI氏・遠藤氏 |
| マーシャルアーツ（請負） | 建物施工、契約クロージング、銀行同行、最終確認 | 三浦部長・長谷川徹・武田常務・長谷川光 |
| お客様 | 買主。住宅ローンの契約者 | — |
| 銀行（信金・地銀・JA） | 住宅ローンを貸す。審査する | 水戸信金・三十三銀行・JA 等 |

**新人のあなたの最初の仕事**は、この流れの中で「日程調整」「お客様への持ち物連絡」「進捗共有」を正確に回すこと。ここを落とさなければ案件は前に進む。

## 新築販売の全体フロー（今どこにいるか）

8ステップで進む。**今どのステップか**を常に意識する。

| # | ステップ | 主役 | 目安 | これができたら次へ |
| :-- | :-- | :-- | :-- | :-- |
| ① | 物件提案 | 東亞地所→客 | 1〜2回 | お客様が「この物件で進めたい」 |
| ② | 事前審査 | 東亞地所→銀行 | 3〜10日 | 銀行から「事前OK」が出た |
| ③ | 売買契約＋本申込 | 両社＋客 | 1日（同日） | 契約書に押印＋本申込を出した |
| ④ | 本申込結果 | 銀行→客 | 7〜10日 | 銀行が正式に「貸します」 |
| ⑤ | 金消契約 | 銀行＋客 | 結果+3〜5日 | ローン契約に押印した |
| ⑥ | 決済（土地・着工金） | 両社 | 金消+3〜5日 | お金が動き、土地着工へ |
| ⑦ | 中間金 | 銀行＋客 | 工事中 | 上棟前後の入金が済んだ |
| ⑧ | 最終金＋引き渡し | 両社＋客 | 竣工後 | 鍵を渡し、入居 |

新人はまず **②事前審査と③契約日の段取り**を完璧に回せるのが最優先。ここで日程・持ち物がズレると全部が後ろにずれる。

## ★ステップ①：物件提案

### 手順

1. お客様の **勤務先・現住所・実家** と、**年収・勤続年数・年齢** を確認する
2. お客様の生活圏に合うエリアで、総額 **4,000〜4,600万** に収まる物件を選ぶ
3. 下の標準フォーマットで提案資料を作る（月々返済額は金利1%で計算して必ず併記）

#### 物件提案フォーマット（このまま埋める）

```
お客様名：◯◯ ◯◯ 様
場所（住所）：都道府県＋市町村＋番地
土地：◯◯㎡ / 約◯◯坪 / ◯◯◯万円
建物：約◯◯坪 / ◯◯◯◯万円
諸費用：300万円（定型）
総額：◯◯◯◯万円
借入期間：40年 or 50年
仲介会社・担当：東亞地所 / ◯◯氏
請負会社：株式会社マーシャルアーツ
月々返済（概算・金利1%）：約◯万円
```

### 考え方

なぜ諸費用は毎回「300万円」なのか → **総額を即計算でき、銀行への打診書式が統一でき、審査が速くなる**から。新人は内訳交渉に時間を使わず、まずこの定型で総額を出せるようにする。

借入期間を40〜50年と長く取るのは、**月々返済額を下げて買える人の間口を広げる**ため。地方の信金・地銀・JAは長期ローンに柔軟。

### 判断基準（迷ったら）

- 総額が4,000〜4,600万に収まらない → 土地/建物の坪数で調整するか、別物件を検討
- 月々返済が10万を大きく超える → 期間を伸ばす or 価格を下げる。それでも無理なら銀行打診前に相談
- お客様の生活圏から遠すぎる → 「なぜそこ？」と審査で疑われる。エリアを見直す

### よくあるミス

- 月々返済額を併記し忘れる（お客様が一番知りたい数字）
- 住所を「市町村まで」で止め、番地を書かず銀行打診で差し戻される
- 請負会社＝マーシャルアーツの記載漏れ

## ★ステップ②：事前審査（仮審査）

事前審査＝銀行が「この人になら貸せそうか」を仮で判断する手続き。**本契約の前にここを通す**のが鉄則。

### 手順

1. お客様の属性を集める：**年齢・年収・勤続年数・勤務先・所有資格**（転職5年未満なら前職の社名・勤続も）
2. エリアに応じて **2〜3行に同時（並列）で打診**する
3. 一番早くOKが出た、または金利・期間が一番有利な条件で進める

#### エリア別 打診先（まずここに出す）

| エリア | 第1候補 | 第2候補 | 第3候補 |
| :-- | :-- | :-- | :-- |
| 茨城 | 水戸信金 | JA | 筑波銀行 |
| 栃木 | 東和銀行 | 烏山信金 | JA |
| 三重・桑名 | 桑名三重信金 | JA | 三十三銀行 |
| 愛知 | 三十三銀行 | JA | 信金 |
| 岐阜 | 岐阜信金 | JA | 三十三銀行 |
| 鹿児島 | 鹿児島相互信金 | JA | — |

### 考え方

なぜ複数行へ同時に出すのか → 1行ずつ待つと時間がかかり、お客様の購入意欲が冷める。**並列で出せば最短・最良条件を選べる**。これはお客様の利益でもある。

借入期間は **完済時年齢80歳まで** で逆算する（例：40歳なら最長40年）。年齢が若いほど50年が使える。

### 判断基準（迷ったら）

- 勤続が短い／転職直後 → 前職含めて勤続年数を説明できるか確認。1行ダメでも他行で通ることがある
- 事前審査がNG → 落ち込む前に **48時間以内に別物件（同エリア・同価格帯）を準備**＝振替物件（後述）

### よくあるミス

- 1行だけに出して結果を待ち、時間を空費する
- 年齢から逆算せず、組めない期間で提案してしまう
- 銀行からお客様に確認電話が入ることを伝え忘れ、お客様が慌てる

### ★お客様への伝え方（このまま使う）

**並列打診のとき**
「条件を比べたいので、◯◯銀行と△△銀行の両方に審査をかけておきますね。お客様にとって一番有利な方を選びましょう。」

**銀行から確認電話が入るとき**
「この後、銀行から内容確認のお電話が入る予定です。もし分からないことがあれば『仲介の担当に確認します』とお伝えいただいて大丈夫です。」

## ★ステップ③：売買契約＋本申込（同日進行）

事前審査がOKになったら、**土地の売買契約**と**住宅ローンの本申込**を同じ日にまとめて行う。お客様の貴重な時間を1回で使うため。

### 手順

1. 売買契約の場所・時間を予約（ファミレス＝ガスト/コメダ等）
2. 本申込の場所・時間を予約（銀行支店）
3. 2会場は **車で5〜10分以内** に設定し、契約→本申込の順で1時間ずらして組む（例：13:00契約→14:00本申込）
4. お客様に持ち物リストを送る／弊社準備書類を揃える
5. 当日：契約書に押印 → 手付金領収書を仲介から受け取る → 銀行で本申込

#### お客様の持ち物（③用・このまま送る）

- 住民票 2通（世帯全員・続柄記載、本籍とマイナンバーは省略）
- 印鑑証明 4通
- 実印、銀行用の届出印（予定印）
- 運転免許証、健康保険証（マイナ保険証 or 資格情報のお知らせ）
- 課税証明 直近分 1通
- 源泉徴収票（あれば）

#### 弊社が準備する書類

- 土地売買契約書
- 工事請負契約書（見積書・間取図を含む）
- 建築確認申請受理証 または 確認済証
- 課税証明書
- 手付金領収書（当日、仲介会社から受け取る）

### 考え方

同日に進めるのは、**お客様の移動と心理的負担を最小化**しつつ案件を一気に一段進めるため。だから会場間の距離（車5〜10分）と時間配分（1時間ずらし）が命。ここを雑にすると遅刻・押印漏れで全部やり直しになる。

### 判断基準（迷ったら）

- 2会場が車10分以上離れる → 時間配分を見直し、余裕を持たせる
- お客様が書類を揃えられない → 揃う日に合わせて契約日をずらす（無理に当日やらない）

### よくあるミス

- 持ち物リストの「通数」を間違える（印鑑証明4通・住民票2通）
- 手付金領収書を仲介から受け取る段取りを忘れる
- 実印と銀行届出印を混同してお客様に伝える

## ★ステップ④：本申込結果

本申込から **7〜10日後**、銀行が正式に「貸します（承認）」を出す。ここで初めてローンが確定する。

### 手順

1. 銀行から結果連絡を受ける
2. 承認なら → ⑤金消契約の日程調整に進む
3. 否決なら → 他行への再打診 or 振替物件を即検討（ステップ②の判断基準へ）

### よくあるミス

- 承認が出た瞬間に気を抜いて、金消の日程調整が遅れる（決済が後ろにずれる）

## ★ステップ⑤：金消契約（きんしょう）

金消契約＝**金銭消費貸借契約＝住宅ローンそのものの契約**。本申込結果から3〜5日後に設定し、決済日はさらに3〜5日後になる。

### 手順

1. 金消日と決済日を確定し、お客様に伝える
2. 持ち物リストを送る
3. 通帳の銀行受取ルール（**決済日から3営業日の17時まで**）をお客様に共有

#### お客様の持ち物（⑤用）

- 運転免許証
- 普通預金通帳 1冊（＋銀行により通知預金通帳）
- 実印・届出印
- 印鑑証明・住民票（銀行が預かる場合あり）

### 考え方

「金消（きんしょう）」は現場で毎日出てくる言葉。意味は**ローン契約**。ここに押印して初めてお金が借りられる状態になる。印鑑証明・住民票を銀行が預かったら、次の手続きで「再発行が必要か」を必ず確認する（無駄な再取得を防ぐ）。

### 判断基準（迷ったら）

- 決済日が **毎月15日前後（年金支給日）** にかかる → 銀行が混み、振込が午後にずれることがある。余裕を見て案内
- 通帳の受取時間をお客様が過ぎそう → 早めにリマインド

### よくあるミス

- 通帳の受取期限（決済日+3営業日17時）を伝え忘れる
- 銀行が預かった書類を、お客様が「もう手元にない」と慌てる（事前説明で防ぐ）

## ★ステップ⑥：決済（土地代・着工金）

決済＝銀行がローンを実行し、土地代金や着工初期費用が支払われる日。ここから**土地の工事（着工）**が始まる。

### 手順

1. 金消から3〜5日後に決済日を設定
2. 土地残代金・固都税精算金・仲介手数料・登記費用・着工金の請求関係を確認
3. 決済完了 → 土地着工

### よくあるミス

- 着工金の請求書類の漏れ（領収書一式を事前に揃える）

## ★ステップ⑦：中間金

土地着工後、**上棟（建物の骨組みが立つ）前後**で中間金を入金する。

### 手順（持ち物・書類）

- お客様の持ち物：運転免許証・通帳1冊・届出印
- 弊社持参：中間金請求書／土地売買時の領収書一式（土地残代金・固都税精算金・仲介手数料・登記費用・着工金）

### よくあるミス

- 上棟のタイミングを現場と共有せず、中間金の入金日がずれる

## ★ステップ⑧：最終金＋引き渡し

建物完成（竣工）後、残金（最終金）を支払い、鍵を渡して入居。お客様はこの前後で**新住所に住所変更**しているので、書類はすべて**変更後の住所**で揃える。

### 手順

#### お客様の準備物（住所変更後）

- 銀行印（実印）
- 住民票 2通（新住所）
- 運転免許証（住所変更後）
- 印鑑証明書 2通（新住所）
- マイナンバーカード（住所変更後）
- 土地権利証 預り証

#### 弊社の準備物

- 火災保険請求書
- 保存登記・表題登記請求書
- 最終金支払金請求書
- 表題登記申請書類
- 変更後の住所・地番が確認できる書類

### 仕上げ（これで完了）

- 新居の鍵・各種保証書類を一式まとめてお客様へ手渡す
- 火災保険会社（インターランド / 03-3231-1200）から契約電話が入る予定を、事前にお客様へ共有
- 最終金の実行日と、**入居可能日（外構工事の完了予定日）**を明示する

### よくあるミス

- 住所変更前の住民票・免許証で書類を作ってしまう（全部やり直し）
- 入居可能日＝外構完了日を伝えず、お客様が引越し日を決められない

## ★毎日の仕事：進捗共有と日程調整

新人が一番よく触るのがここ。**全案件の状況を1メッセージで共有**する文化がある。

### 進捗共有テンプレート（週1〜2回投下）

```
■◯◯様 - 外構工事 / お引越し ◯月◯日
■◯◯様 - 最終金候補日 ◯月◯日〜
■◯◯様 - 金消候補日 ◯月◯日 or ◯日
■◯◯様 - 本申込結果 ◯月◯日予定
■◯◯様 - 事前審査中（◯◯銀行）
■◯◯様 - 事前準備中（資料待ち）
■◯◯様 - 物件提案 ◯月◯日（日）13:00〜
```

共有が来たら、マーシャル側は日付・お客様名を確認して「**畏まりました！**」で受ける。

### 日程変更が来たときの対応速度（鉄則）

| 経過時間 | やること |
| :-- | :-- |
| 30分以内 | 「承知しました」の第一報を返す |
| 3時間以内 | 第二・第三希望の候補日を出す |
| 24時間以内 | 銀行・関係先との再調整を完了 |
| 即日無理なら | 「明日朝一でご連絡します」と先回り |

### 標準フレーズ集（LINEはこれで足りる）

| 場面 | フレーズ |
| :-- | :-- |
| 冒頭の挨拶 | お世話になっております！ |
| 指示を受けた | 畏まりました！ / 承知致しました！ |
| お礼 | ありがとうございます！ / とんでもございません！ |
| 共有を受けた | ご共有ありがとうございます！ |
| 確認する | 念のためご確認させていただきます |
| お願い | お忙しい中恐れ入りますが、ご確認の程よろしくお願いいたします |
| 締め | 引き続き宜しくお願い致します！ |
| 夜分・土曜の連絡 | 夜分遅くに申し訳ございません |

## ★トラブル対応（初心者がまず覚える3原則）

1. **即日返答** — 24時間放置しない。まず「受け止めました」を返す
2. **代替案を3つ** — ただ謝らず、必ず次の候補（日程/物件）を3つ出す
3. **謝意を先に** — お客様の都合を優先する姿勢を示してから業務調整に入る

### よくある場面と動き方

| 場面 | 動き方 |
| :-- | :-- |
| 事前審査がNG | 48時間以内に別物件（同エリア・同価格帯）＝振替物件を提案。他行（JA等の長期対応）も検討 |
| お客様が体調不良・予定変更 | 体調を最優先し共感を一言。候補日を3つ出す。銀行へ即日日程変更連絡 |
| 銀行担当者が異動 | 新担当の連絡先を確認。共有事項に「担当変更」を明記。経緯を新担当へ説明 |
| 工期が遅れた | 即日一次連絡＋謝意。代替候補日3つ。引き渡し・引越し日への影響を計算して提示 |

### 振替物件のやり方（重要）

事前審査がNGでも、お客様を逃さないのが東亞地所モデルの強み。

1. NG確定後 **48時間以内** に別物件を準備
2. **同エリア・同価格帯・同間取り** を優先
3. 伝え方：「振替で申し訳ございません」と謝意を示しつつ、新提案は前向きに見せる
4. 内見が必要なら、同エリアの既存物件（他のお客様邸）を案内に使う
