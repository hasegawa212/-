# 東亞地所案件 学習ドリル

*一問一答＋確認テストで「勉強できる」教材*

| 項目 | 内容 |
| :-- | :-- |
| 発行 | 株式会社Martial Arts |
| 対象 | 東亞地所案件をこれから覚える人（新人・自習用） |
| 目的 | 『東亞地所モデル_新築販売戦略マニュアル』『かんたん進行手順書』の内容を、解いて覚える |
| 使い方 | ①一問一答で覚える → ②確認テストで力だめし → ③巻末解答で答え合わせ |

> **▍勉強の進め方（3ステップ）**
> 1. **第1部 一問一答** … 章ごとに上から解く。答えは隠して、まず自分で言ってみる。
> 2. **第2部 確認テスト** … 章末テスト→総合テストの順。8割とれたら合格。
> 3. **第3部 解答** … 答え合わせ。間違えた問題は一問一答に戻って覚え直す。
> 目安: 1日1章×8日でひと通り。1週間後にもう一度確認テストだけ解くと定着する。

## 第1部 一問一答（章ごとに覚える）

## 1章 全体像と8ステップ

1. 東亞地所案件は、全部で「____ステップ」で進む。
2. 8ステップを順番に並べよ：物件提案 → ____ → 売買契約＋本申込 → 本申込結果 → ____ → 決済(土地・着工金) → 中間金 → ____。
3. 「事前審査」は1案件につき、____〜____行に同時(並列)で打診するのが標準。
4. 売買契約と本申込は「____日（同じ日）」に進めるのが標準である。◯か×か。
5. 金消契約は本申込結果が出てから「____〜____日後」に設定する。
6. 決済日は金消契約から「____〜____日後」となる。
7. 中間金を入れるのは、工事の「____前後」のタイミングである。
8. 最終金＋引き渡しは、建物が「____」したあとに行う。

## 第2部 確認テスト（力だめし）

## 確認テストA（基礎・穴埋め）

A-1. 8ステップを順に書け（①〜⑧）。
A-2. 諸費用の固定額・総額レンジ・借入期間・想定金利の4つの標準値を書け。
A-3. 売買契約と本申込を同日に行うとき、2つの会場の距離はどれくらいにするか。
A-4. 金消→決済の標準的な日数差を書け。
A-5. エリア別の第一候補銀行を3つ書け（茨城／栃木／三重・桑名）。

## 確認テストB（◯×）

B-1. 事前審査は最も早く承認が出た1行だけに絞って打診する。（◯／×）
B-2. 諸費用は案件ごとに細かく見積もって最適化する。（◯／×）
B-3. お客様都合の日程変更は、翌営業日にまとめて再調整すればよい。（◯／×）
B-4. 長期50年のローンは、JAが対応しやすい。（◯／×）
B-5. 最終金時の書類は、住所変更前のものでよい。（◯／×）

## 確認テストC（記述・実務）

C-1. 事前審査がNGになった。あなたが取るべき行動を、順を追って3つ書け。
C-2. お客様が体調不良で面談をキャンセルした。最初の30分・3時間・24時間で何をするか。
C-3. 勤続半年（転職直後）のお客様。否決を避けるために、どの銀行系統で・どんな補強書類を用意して出すか。
C-4. 「銀行から確認電話が入る」とお客様に伝えるとき、何を添えて伝えるか。

## 確認テストD（ロールプレイ課題）

D-1. 物件提案フォーマットを使って、架空の物件（茨城・総額4,200万）の提案文を1件作成せよ。
D-2. LINE進捗共有フォーマットで、3名のお客様（事前審査中／金消候補日／引き渡し）の進捗を1メッセージにまとめよ。

## 第3部 解答

## 1章 解答
1. 8 / 2. 事前審査、金消契約、最終金＋引き渡し / 3. 2〜3 / 4. ◯（同日進行が標準） / 5. 3〜5 / 6. 3〜5 / 7. 上棟 / 8. 竣工（完成）

## 2章 解答
9. 東亞地所 / 10. マーシャルアーツ / 11. 請負 / 12. 仲介

## 3章 解答
13. 300 / 14. 4,000〜4,600 / 15. 40〜50 / 16. 1 / 17. 420〜500 / 18. 8〜10 / 19. 例：瞬時に総額が計算できる／銀行打診の書式が統一でき審査が早い／見積もり工数が減る／内訳交渉に時間を取られない（いずれか）

## 4章 解答
20. 信用金庫、JA / 21. JA / 22. 水戸信金 / 23. 東和銀行 / 24. 桑名三重信金 / 25. 金利等の条件

## 5章 解答
26. 4 / 27. 2 / 28. 手付金 / 29. 普通、1 / 30. 住所

## 6章 解答
31. 畏まりました（承知致しました） / 32. 30 / 33. 3 / 34. 24 / 35. 進捗

## 7章 解答
36. 80 / 37. 勤続、住所 / 38. 購入 / 39. 在籍 / 40. 就業条件

## 8章 解答
41. 48 / 42. 振替 / 43. ①即日返答 ②代替案 ③謝意先行 / 44. 3、17 / 45. 15

## 確認テストA 解答
A-1. ①物件提案 ②事前審査 ③売買契約＋本申込 ④本申込結果 ⑤金消契約 ⑥決済(土地・着工金) ⑦中間金 ⑧最終金＋引き渡し
A-2. 諸費用=300万円／総額=4,000〜4,600万円／借入期間=40〜50年／想定金利=1.0%前後
A-3. 車で5〜10分以内（10分以内）
A-4. 金消契約から3〜5日後に決済
A-5. 茨城=水戸信金／栃木=東和銀行／三重・桑名=桑名三重信金

## 確認テストB 解答
B-1. ×（2〜3行に並列で打診し、最短承認 or 最有利条件で進める）
B-2. ×（300万円で固定するのが標準）
B-3. ×（30分以内に第一報、3時間以内に候補日、24時間以内に再調整）
B-4. ◯
B-5. ×（住所変更後の書類が必要）

## 確認テストC 解答（例）
C-1. ①48時間以内に同エリア・同価格帯・同間取りの別物件を準備 ②「振替で申し訳ございません」と謝意を示し新提案を前向きに見せる ③JAなど長期対応の銀行で再打診できないか検討。
C-2. 30分以内＝体調を気づかう受け止めの第一報／3時間以内＝次回候補日を3つ以上提示／24時間以内＝銀行・関係先へ日程変更を連絡。
C-3. 東和・大光など中堅地銀／信金へ。転職理由を前向きに明文化し、在籍証明書・就業条件明示書・購入動機書で補強。免許の現住所を更新し源泉と整合を取る。
C-4. 質問内容を予告し、「分からなければ仲介の営業に確認しますとお伝えください」と添える。
