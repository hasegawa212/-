## 一行サマリー（4資料の結論）

- **入口（アポ）**: 6月から反響(bookrun)自動予約が稼働し新規アポ供給が約3倍。品質もC偏重→B増加。
- **中身（商談）**: 成約系は中央値**7回目**の面談で発生。短期決着なし＝**切らさず運ぶ**のが本質。
- **数字（AI基準）**: 総支出を年収帯の安全レンジに**文面で握る**／DTI≤33%／手元現金≥150万死守／GO・STOPを数値で判断。
- **出口（クロージング）**: 契約・本申込が最大の伸びしろ。**決断取り＋上席同行の活用**が次の一手。
- **見切り**: 温度感ゼロ・転勤確定・属性NGは早期打ち切りが正解（打ち切りの半分は1回目）。

## 元データと注意

- 元データ: 社内Slack `martial-arts-ghd` の各報告チャンネル（2026年5〜6月中心）。
- 顧客の個人特定情報（氏名・住所・電話・メール）は全資料で**マスキング**。手法・型・数値基準のみ教材化。
- 定量値は取得時点（2026/6/11）の参考値。サンプル数・期間の限界は各資料内に明記。
