> 最終更新: 2026-05-26
> 目的: 業務で繰り返し使用するファイル雛形・記入例の所在を一元管理する

---

## 1. 取扱方針

このディレクトリは **テンプレート（雛形）のインデックス** です。

- 実体ファイル（Excel / PDF）は **Google Drive** で管理（git 管理外）
- このフォルダにはテンプレ名・用途・所在・記入ルールだけを記録
- バージョン管理が必要なテンプレは別途方針を決める

理由: テンプレ自体は社外秘 PII を含まないが、**記入後のテンプレは顧客 PII を含むため git にコミットしてはならない**

## 3. テンプレ新規登録ルール

新しいテンプレを社内で使い始めるとき：

1. Drive の `09_資料・アーカイブ/テンプレート/` 配下にアップロード
2. 本 README に行を追加（テンプレ名・種別・用途・Drive 上のファイル名）
3. テンプレを使う **業務マニュアル**（業務マニュアル/配下）にリンクを追加

---

## 4. 機微情報の取り扱い

- **空のテンプレ**は社外秘 PII を含まないため git で管理してよい（ただし本リポジトリでは Drive 管理を方針）
- **記入後の書類**は顧客 PII を含むため、**絶対に git にコミットしない**
- 顧客記入済み書類は Drive の **アクセス制限フォルダ**（CEO + 主担当 + バックアップのみアクセス可）に保管
