# 売上利益管理（顧客名・売上・利益の抽出シート）

アップロードされた以下の元資料から、**顧客名・売上・利益（粗利）** を中心に
見やすく整形・集約したシートです。

## 成果物

| ファイル | 内容 |
| --- | --- |
| `売上利益サマリー.xlsx` | 整形済みワークブック（3タブ）。配色・見出し固定・桁区切りあり |
| `月次サマリー.csv` | 第3期〜第6期の月次売上（不動産/その他/合計/累計/前月比/主要顧客）。6期は2026/01〜06を網羅 |
| `顧客別明細.csv` | **第4期〜第6期**の案件別 顧客名・売上・粗利（物件巾）・**利益**・種別・支社・状況 |
| `契約報告_2026.csv` | 2026/03/22〜04/19 の契約報告一覧（お客様名・種別・担当 ほか） |

> **利益列**: 第4期〜第6期（2024/01〜2026/06）の全月を、各月『売上表』PDF の顧客別利益から抽出。
> 月別利益合計はPDFの累計マーカーと一致（例: 2025/12=7,955万 / 2026/02=2,994万 / 05=3,067万 / 06=3,257万）。
> 第3期（2022年）は元PDFの該当ページが部分的・空欄のため明細化していません（月次サマリーには集計あり）。
> 5〜6期は売上＝融資価格・支社/銀行/状況を案件台帳から補完。私募債/物販など一部の補助項目はPDF自動抽出のため要スポット確認。

### `売上利益サマリー.xlsx` のタブ構成
1. **月次サマリー** … 期 / 年月 / 不動産売上 / その他売上 / 月合計 / 累計 / 前月比 / 主要顧客 / 確定状況（進行中の月は黄色）
2. **顧客別明細** … 期 / 年月 / 顧客名 / 売上 / 粗利（物件巾） / 支社 / 銀行・売主 / 状況 / タイプ（5期=緑・6期=青）
3. **契約報告_2026** … 直近の契約報告（お客様名・報告種別・担当・次回予定・備考）

> 金額単位は **万円**。`売上` は元台帳の融資価格、`粗利（物件巾）` は元台帳の物件巾を採用。
> 純利益（歩合控除後）は元資料に項目が無いため算出していません。
