## スコアリング方法

### 「不安・孤立型」離脱リスク（メインモデル）

仕事や人間関係でストレスを抱え、孤立して退職するタイプ。一般的な早期離職パターン。

```
不安・孤立型リスクスコア
= max(0, 50 - 精神耐性)         ← 耐性低いほど加点
+ max(0,  不適応傾向 - 30)        ← 不適応高いほど加点
+ max(0, 40 - 共感力)            ← 共感低いと孤立しやすい
```

### 「不満・抗議型」離脱リスク（補助モデル）

成果や報酬に強い不満を持ち、自己主張的に離脱するタイプ。**石井専務のパターン**。

```
不満・抗議型リスクスコア
= max(0, 固執性 - 50)            ← 固執強いほど自分の理屈で動く
+ max(0, 金権志向 - 50)          ← 金銭への期待が大きい
+ max(0, 50 - 安定志向)           ← 安定よりも変化を求める
+ max(0, 50 - 共感力)            ← 組織への協調性が低い
```

参考：石井光静氏の不満・抗議型スコア = (73-50) + (63-50) + (50-29) + (50-39) = **68**  
→ メインモデル（不安・孤立型）では 15 と低かったが、抗議型では **68（極高）** だった。**バックれを Scouter 単独で予測できた**ことになる。
