## スコアの読み方

Scouter 検査SS のスコアは標準化得点（おおむね平均50・標準偏差10）。20〜80 のレンジで、

- **40〜60 がボリュームゾーン（中央 68.2%）**
- 60 超 = 上位 15.9%、70 超 = 上位 2.3%
- 39 未満 = 下位 15.9%、30 未満 = 下位 2.3%

「高いほど良い」尺度（主体性・耐性・職務適性・能力系）と、「低いほど良い」尺度（ネガティブ傾向・虚偽傾向）が混在するため、各ファイルの注記を参照。
