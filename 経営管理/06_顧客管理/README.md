# 06_顧客管理

顧客に関する情報・契約・サポート履歴を一元管理する区分です。

## サブフォルダ構成

| フォルダ | 用途 | 現状 |
| --- | --- | --- |
| `顧客情報/` | 顧客の基本情報（連絡先・属性・取引履歴サマリーなど） | 未整備 |
| `契約書/` | 契約書原本および契約条件を一覧化した台帳・サマリー | `2026-05-29_顧客契約一覧.csv` ほか（[README](契約書/README.md)） |
| `サポート履歴/` | 問い合わせ・対応履歴 | 未整備 |

## 関連データ

- `契約書/2026-05-29_顧客契約一覧.csv` — 顧客契約台帳（5 件）
- `契約書/2026-05-29_契約サマリー.md` — 上記の集計サマリー（満了・更新管理）
- リポジトリ直下 `テレアポ管理シート.csv` / `経営管理/03_営業/テレアポ/` — 見込み客のテレアポ管理（成約後の顧客情報は本区分へ）

## 運用ルール

- ファイル名は `YYYY-MM-DD_概要.ext` 形式（`経営管理/README.md` の命名規則）に準拠する。
- 個人情報を含むため、取り扱いは `経営管理/README.md` の運用ルールに従う。
- 分類に迷う場合は安易に新規フォルダを作らず、上記 3 サブフォルダ配下にサブフォルダを切る。
