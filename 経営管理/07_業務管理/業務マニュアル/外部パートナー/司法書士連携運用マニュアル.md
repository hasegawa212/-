---

## 1. このマニュアルの目的

司法書士は決済時の**最重要パートナー**（登記・本人確認・書類管理を一手に担う）。  
ここでミスると：
- 同日決済の所有権移転が遅れる
- 登記順位が変わる
- 振込ミスで信用毀損
- 顧客クレーム

→ 司法書士情報の属人化を解消し、誰が依頼しても同じ品質で連携できる状態を作る。

| 7 | 直通電話・携帯 | 必須 |  |

## 3. 司法書士マスタの保管場所

⚠️ **個人情報（個人名・連絡先・口座）を含むため、git にコミットしない**。

[銀行マスタ](../案件創出/ローン付け/銀行マスタ運用ルール.md) と同じ場所（CRM / Google Sheets / Notion）で**外部パートナーマスタ**として一元管理。

## 4. 案件への司法書士アサインフロー

```
案件決済日確定の見込みが立った時点（T-30 日目安）
   ↓
司法書士マスタから対応可能事務所を選定
   ├─ 法務局管轄に対応しているか
   ├─ 決済日のスケジュール空き
   ├─ 案件特性（中間省略・相続等）への適性
   └─ 過去取引の関係性
   ↓
1 事務所に絞ってアサイン依頼（電話 + メール）
   ↓
受領確認 → 必要書類リスト・費用見積もり受領
   ↓
案件管理台帳に司法書士情報を紐付け
   ↓
[住宅ローン決済日対応マニュアル] のタイムラインへ
```

## 6. 決済前 司法書士連携チェックリスト

### T-21 日（決済 3 週間前）
- [ ] 司法書士アサイン完了
- [ ] 費用見積もり受領
- [ ] 必要書類リスト受領

### T-14 日
- [ ] お客様への必要書類案内（住民票・印鑑証明 通数含む）
- [ ] 売主側への必要書類確認
- [ ] 銀行への抵当権設定書類確認

### T-7 日
- [ ] 全関係者に決済日時・場所のリマインド
- [ ] 司法書士費用 振込先確認（請求書原本受領）
- [ ] 書類最終チェック（売主・買主・銀行・当社）

### T-3 日
- [ ] お客様 持参物 最終リマインド
- [ ] 司法書士から書類完備確認の連絡

### T-1 日
- [ ] 全関係者リマインド
- [ ] 当日タイムスケジュール再確認

## 7. 司法書士費用 支払いルール

### 振込ミス防止
- [ ] 請求書**原本**を受領（口頭・メールのみは不可）
- [ ] 受取人正式名称確認（個人名 / 法人名 / 屋号）
- [ ] 振込前 **2 名ダブルチェック**（担当 + 上長）
- [ ] 振込控えを案件ファイルに保管
- [ ] 振込完了を司法書士に連絡

### 支払いタイミング
- 標準：**決済当日に振込**（買主資金から）
- 例外：当社案件で当社が支払主の場合、決済翌営業日まで

## 8. 当日 段取り（司法書士との連携部分）

詳細は [住宅ローン決済日対応マニュアル §9](../決済対応/住宅ローン決済日対応マニュアル.md) を参照。  
このマニュアルでは**司法書士との連携部分のみ**抜粋：

```
10:00  関係者集合
10:05  司法書士による本人確認（免許証等）
10:15  司法書士主導で書類記入・押印
       - 金消契約書
       - 登記委任状
       - 抵当権設定書類
10:30  司法書士が書類完備を確認 → 完了サイン
10:50  司法書士へ費用振込
11:00  司法書士が法務局へ移動 or 当日オンライン申請
11:30  鍵引渡し（同日引渡し案件）
```

⚠️ 司法書士が「**当日中に法務局申請完了**」を保証する事務所を選ぶ。  
翌日申請になると登記順位リスクあり。

## 9. 決済後フォロー

### 司法書士からの完了連絡
- [ ] 登記完了確認（数日〜2 週間後）
- [ ] 権利証（登記識別情報通知）の発行・送付タイミング確認
- [ ] お客様への到着フォロー

### 案件ファイルへの記録
- [ ] 司法書士費用 領収書
- [ ] 登記完了証
- [ ] 権利証発行通知
- [ ] 司法書士マスタ「過去取引履歴」に追記

## 10. トラブル時の対応

| トラブル | 対応 |
| --- | --- |
| 司法書士の急病・不在 | 同事務所内の代替者対応依頼 → なければ別事務所緊急アサイン |
| 書類不備で当日決済延期 | 売主・買主・銀行に即時連絡、リスケ |
| 振込先 / 金額ミス | 銀行へ即時組戻し依頼、司法書士へ謝罪 + 再振込 |
| 登記完了が遅延 | 司法書士から原因 + 復旧見込み確認、お客様連絡 |
| 司法書士の過誤による登記ミス | 司法書士の責任 → 賠償交渉、信用毀損なら継続採用見直し |

## 11. 司法書士の評価・継続採用判定

四半期ごと（4 月・7 月・10 月・1 月）に以下を評価：

| 評価軸 | 5 点満点 |
| --- | --- |
| 対応スピード | 連絡〜回答までの早さ |
| 書類正確性 | ミスの少なさ |
| お客様対応 | 丁寧さ・分かりやすさ |
| 緊急時対応力 | 急な依頼への柔軟性 |
| 費用妥当性 | 相場との比較 |

合計 20 点以上：継続採用 / 15〜19 点：要観察 / 14 点以下：別事務所への切替え検討
