## 1. このサービスの位置づけ

住宅ローンを利用したお客様向けの**無料・任意**の保険見直しサービス。

- **無料**：ローン顧客向け付帯サービスとして提供
- **任意**：強制ではなく、希望者のみ
- **目的**：団信加入により発生する**保障の重複を整理**し、保険料を最適化
- **収益モデル**：保険代理店としての継続手数料収入（お客様負担なし）

---

## 2. 案内タイミング

### 推奨：決済日 当日 顔合わせ
- お客様・司法書士・銀行担当が揃う場で短時間（10〜15 分）
- **詳細相談は別途設定**（20〜30 分の専用枠）

### 代替：決済後 1〜2 週間
- 引越し落ち着いた頃に電話 / 面談アポ

## 3. 初回案内の標準スクリプト（10 分版）

### Step 1：自己紹介・サービスの位置づけ（1 分）

```
本日はお時間いただきありがとうございます。
私、[保険代理店名] の [担当者名] と申します。

住宅ローンをご利用いただいたお客様向けに、保険の見直しサービスを
無料でご提供しております。本日は概要のご案内と、ご希望の確認まで
させていただければと思います。
```

### Step 2：なぜ住宅ローン契約時に保険見直しが有効か（3 分）

```
住宅ローンには団体信用生命保険（団信）が組み込まれており、
万が一の場合はローン残債が完済される仕組みです。

つまり、ご家族には「ローンのない住宅」が残ることになります。

これは、すでにご加入の生命保険で同じ保障を持っている場合、
保障が重複している状態と言えます。

重複部分を整理すると、保険料を抑えながら必要な保障を維持できる
可能性があります。
```

### Step 3：見直しの進め方（3 分）

```
具体的には、次回 20〜30 分のお時間で、以下を確認します：

1. 現在ご加入中の保険内容の整理
2. 団信との重複箇所の特定
3. 必要保障額の試算（お客様のライフプランに合わせて）
4. 見直し案のご提案（複数パターン）

ご提案を採用するかどうかは、もちろんお客様のご判断です。
```

### Step 4：意向確認（2 分）

```
ご興味がおありでしたら、次回お打ち合わせの日程を決めさせて
いただきたいのですが、いかがでしょうか？

候補日：[2 月後半・午前中目安]
所要時間：20〜30 分
場所：ご自宅 / 弊社 / オンライン いずれも可

ご都合のよろしい方法でお願いいたします。
```

### Step 5：次回 持参物のお願い（1 分）

```
次回までに、以下のものをご準備いただけますと、より具体的な
ご提案ができます：

【できれば持参】
- 現在ご加入中の保険証券
- 設計書（あれば）
- 控除証明書（あれば）

【メモで可】
- 預貯金の概況
- 投資経験の有無
- 月々の家計の余裕額
```

## 4. 初回案内 トーク NG 集

| ❌ NG | 理由 |
| --- | --- |
| 「絶対に保険料が下がります」 | 結果保証はできない |
| 「今の保険は無駄です」 | 既契約の否定は信頼を損なう |
| 「すぐに解約しましょう」 | 拙速な提案は後でクレームに |
| 「他社よりお得です」 | 商品比較は次回詳細時に |
| 「弊社の商品を…」 | 初回は中立姿勢を保つ |
| 商品名・金額の具体話 | 詳細は次回（情報整理後） |

## 7. 必要保障額 試算の標準フレーム

```
必要保障額 = ① 遺族の生活費 + ② 住居費 + ③ 教育費 + ④ 葬儀費等
           - ⑤ 遺族年金 - ⑥ 既加入保障 - ⑦ 預貯金・運用資産
```

- ① 月生活費 × 配偶者の平均余命 × 0.7
- ② 団信加入済なら 0（家賃想定なら別途）
- ③ お子様 1 人あたり 1,000 万円 ± 進学プラン
- ④ 200〜300 万円
- ⑤ 公的年金シミュレータで概算
- ⑥ 既加入の死亡保障合計
- ⑦ 預貯金 + 換金可能資産

→ **「① + ③ - ⑤ - ⑥ - ⑦」がプラスなら追加保障検討、マイナスなら過剰**

## 8. 重複保障 整理パターン

| 既加入の特性 | 推奨アクション |
| --- | --- |
| 高額な定期保険（住宅ローン目的） | 減額 or 解約 |
| 終身保険 | 維持（相続対策・葬儀費等） |
| 医療保険 | 維持（団信と無関係） |
| がん保険 | 維持（団信と無関係） |
| 学資保険 | 維持（目的別保障） |
| 個人年金 | 維持（老後資金、性質が違う） |

## 9. 情報取扱いの注意事項

### 個人情報保護
- 同意書なしに保険情報の収集・利用はしない
- 提案後に成約しなかった場合の情報破棄方針を明示

### 適合性原則
- お客様の年齢・収入・知識・経験・目的に**適合した提案**のみ行う
- 投資性商品の勧誘時は特に注意

### 苦情対応
- 初回案内で「強引な勧誘」と受け取られないよう、**「ご判断はお客様」**を 3 回以上明言

## 10. 営業 KPI（運用後 計測項目）

- 決済お客様への案内実施率（目標：95%）
- 次回面談アポ取得率（目標：60%）
- 詳細相談 → 提案実施率（目標：90%）
- 提案 → 成約率（目標：40%）
- お客様満足度（NPS、アンケート）

→ KPI は導入 3 ヶ月後に見直し、現実的な水準にチューニング。
