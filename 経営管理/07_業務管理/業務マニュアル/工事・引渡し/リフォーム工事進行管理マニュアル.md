# リフォーム工事 進行管理マニュアル v1.0（暫定）

> 作成日: 2026-05-16
> 主担当: **★ 要任命**（工事監督者 — 社内に確定担当者なし）
> バックアップ担当: **嶺田 勝**（決済までの段取り責任を持つため）
> 暫定運用: 各案件で都度 工事監督担当を指名（営業担当 or 専任 if 配置可）
> 派生元: 5/15 仕入れ会議議事録（小泉・大崎・平松案件記述）
> 脱属人化スキーム該当原則: ②知識の文書化

---

## 1. このマニュアルの対象業務

決済完了後、引渡しまでの**リフォーム工事の進行管理**。

リフォーム込み販売の場合、決済 → 工事完了 → 引渡しまでの **約 1 ヶ月** が標準業務領域。  
この期間に：
- 工事品質の管理
- 工程の進捗管理
- お客様への進捗共有
- 引渡し準備

---

## 2. 標準タイムライン（決済〜引渡し）

```
[T=決済日]  鍵受領、業者へ工事指示書交付
   ↓
[T+1〜3日]  着工
   ├─ 解体（必要時）
   ├─ 水回り工事
   ├─ 壁・床仕上げ
   ├─ 設備設置
   └─ クリーニング
   ↓
[T+20〜25日] 工事完了、社内検査
   ↓
[T+25〜28日] お客様 内覧（手直し対応）
   ↓
[T+28〜30日] 引渡し（鍵渡し）
```

→ **標準 1 ヶ月。延びた場合はお客様の入居計画に影響**するため期限管理は厳格に。

---

## 3. 工事内容の標準パターン

| パターン | 工事範囲 | 工期目安 | 費用目安 |
| --- | --- | --- | --- |
| A) 軽微 | クリーニング + 部分補修 | 5〜10 日 | 30〜50 万円 |
| B) 標準 | 水回り + 内装 | 15〜25 日 | 100〜200 万円 |
| C) フル | スケルトン + 全面 | 30〜45 日 | 300〜500 万円 |
| D) 特殊 | 増改築 + 外構 | 45 日〜 | 500 万円〜 |

→ **B 標準パターンが多い**。販売時の総額に含めて提示するのが定石。

---

## 4. 業者管理（基本ルール）

### 業者選定の優先順位
1. **過去取引で品質確認済の業者**
2. 工期遵守実績がある業者
3. 価格妥当性
4. 連絡レスポンス速度

### 業者管理 NG
- ❌ 1 案件 = 複数業者の同時進行で詳細指示が拡散
- ❌ 価格だけで初回業者を採用（品質リスク）
- ❌ 業者まかせで現場確認しない（脱属人化原則 ③ 違反）

---

## 5. 進捗管理 チェックポイント

| タイミング | アクション | 担当 |
| --- | --- | --- |
| 着工日 | 業者から着工確認 | 工事監督 |
| 解体完了時 | 現場確認（隠れた瑕疵チェック） | 工事監督 |
| 中間時 | 現場確認 + 写真記録 | 工事監督 |
| 完了 3 日前 | 業者から完了見込み連絡 | 業者→工事監督 |
| 完了日 | **社内検査**（チェックリスト使用） | 工事監督 + 嶺田 |
| お客様内覧前 | 最終クリーニング確認 | 工事監督 |

---

## 6. 社内検査チェックリスト（完了時）

工事完了時、お客様内覧前に必ず実施：

### 全般
- [ ] 工事指示書通りの内容になっているか
- [ ] 設備の動作確認（電気・水道・ガス・空調）
- [ ] 鍵・取手・建具の動作確認
- [ ] 照明の点灯確認

### 水回り
- [ ] 給湯器の動作確認
- [ ] 水漏れ・排水詰まりのチェック
- [ ] 換気扇の動作確認

### 内装
- [ ] 壁紙のシワ・剥がれ
- [ ] フローリングのキズ・浮き
- [ ] 巾木・廻り縁の処理
- [ ] 建具の開閉・収まり

### クリーニング
- [ ] 床・壁の汚れ
- [ ] 水回りの水垢
- [ ] 窓・サッシ
- [ ] 換気扇内部

### 安全
- [ ] 火災報知器の電池・動作
- [ ] 鍵の引渡し本数確認

→ **2 名以上で確認**（脱属人化原則 ①）。記録を写真付きで保管。

---

## 7. お客様内覧の段取り

### 内覧前
- [ ] お客様にアポ取得（引渡し 2〜3 日前）
- [ ] 持参物案内（鍵受領用、印鑑等）

### 内覧時
- [ ] 工事監督が同行
- [ ] **設備の使い方説明**（給湯器・空調・換気扇等）
- [ ] 鍵の取扱い（本数・スペアキー）
- [ ] 緊急時連絡先の案内
- [ ] **指摘事項の即時記録**（写真 + メモ）

### 内覧後
- [ ] 指摘事項を業者へ連絡（48 時間以内）
- [ ] 手直し完了確認
- [ ] お客様へ完了報告 → 引渡し日確定

---

## 8. 引渡し時の引継ぎ書類

お客様にお渡しする標準セット：

- [ ] 鍵（本数明記）
- [ ] 設備取扱説明書（給湯器・空調・換気扇・コンロ等）
- [ ] 保証書（業者保証 + メーカー保証）
- [ ] 緊急時連絡先一覧（業者・当社・管理会社等）
- [ ] **生活ライフライン手続き案内**（電気・ガス・水道の再契約）
- [ ] オートロック・宅配ボックス等の使い方
- [ ] ゴミ出しルール（自治体・物件規約）
- [ ] アフターフォロー窓口の案内

⚠️ 議事録で言及あった **電気・水道再契約案内**、**オートロック開け方共有**は引渡し時に必須。

---

## 9. アフターフォロー（引渡し後）

| タイミング | アクション |
| --- | --- |
| 引渡し 3 日後 | 不具合の有無確認（電話 / LINE） |
| 引渡し 1 週間後 | 入居状況確認 |
| 引渡し 1 ヶ月後 | アフター点検（業者保証範囲内の不具合チェック） |
| 引渡し 半年後 | フォロー連絡 |
| 引渡し 1 年後 | 1 年点検 + 紹介依頼 |

→ アフターフォローが**次案件・紹介につながる**。手抜きしない。

---

## 10. 工事遅延時の対応

工事完了見込みが当初予定より**3 日以上遅延**する場合：

1. **業者から原因 + 復旧見込み**の連絡を取る
2. お客様へ**即時連絡**（黙っているとクレーム化）
3. 引渡し日を再設定（できれば代替案 2 つ）
4. 影響が大きい場合（お客様の引越し業者キャンセル料等）は補填協議
5. 社内会議で**業者の継続採用可否**を判断

---

## 11. 工事費・支払い管理

- 業者への支払い：着手金 / 中間金 / 完了金（業者ごとに条件確認）
- 支払いは**完了検査 + お客様内覧合格後**が原則
- 一括前払いは避ける（業者倒産リスク・品質保証強制力低下）
- 支払い記録を案件ファイルに保管（後の収支管理用）
