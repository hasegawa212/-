## 1. このマニュアルの対象業務

住宅ローン本承認後、**金消契約・決済・登記・引渡し**の一連の手続き対応。

関係プレイヤー：
- お客様（買主）
- 売主
- 金融機関（融資実行担当）
- 司法書士（登記・本人確認）
- 当社（仲介・調整）
- 保険代理店（団信以外の保険見直し提案）

---

## 2. 決済日 標準タイムライン

```
[T-30日]  本承認  →  金消日程確定
[T-14日]  住民票・印鑑証明 必要通数確定
[T-7日]   司法書士費用 振込先確認 / 必要書類最終チェック
[T-3日]   お客様に持参物リスト最終連絡
[T-1日]   全関係者リマインド
[ T 日 ] 決済 = 融資実行 + 売買代金支払 + 登記書類預け
[T+1日]  司法書士が法務局へ登記申請
[T+7日]  既存口座の住所変更等 銀行手続き
[T+30日] 権利証（登記識別情報通知）が新居宛に書留到着
```

## 4. 住民票・印鑑証明 通数管理ルール

**よくある失敗**：通数不足で当日 or 後日追加取得が発生。

| 用途 | 必要通数（標準） |
| --- | --- |
| 所有権移転登記 | 住民票 1 通 |
| 抵当権設定登記 | 印鑑証明 1 通 |
| 銀行 金消契約 | 印鑑証明 1〜2 通 |
| 銀行 口座住所変更 | 住民票 1 通 |
| その他予備 | 住民票・印鑑証明 各 1 通 |
| **合計目安** | **住民票 3 通・印鑑証明 3〜4 通** |

→ **T-14 日に司法書士・銀行に最終通数確認**し、お客様へ取得依頼。

## 5. 金利特約 説明テンプレート（お客様向け）

「変動金利」案件の場合の標準説明：

```
■ 金利の仕組み
- 基準金利：[X.XX]%
- 優遇後金利：[Y.YY]%（適用金利）
- 適用期間：完済まで継続

■ 金利見直しサイクル
- 基準金利の見直し：年 2 回（4 月 / 10 月）
- 月々返済額への反映：見直しから 3 ヶ月後

■ 5 年ルール
- 金利が変動しても 5 年間は月々返済額を変更しない
- 元金と利息の配分で調整
- 5 年後に返済額を再計算（前回の最大 1.25 倍まで）

■ 注意点
- 金利上昇局面では、5 年後に返済額が増える可能性
- 元利配分が利息寄りになると、元金返済が進みにくい
- ライフプラン上の余裕資金で繰上返済を検討すると効果的
```

## 6. 司法書士費用 振込ミス防止

**よくある事故**：振込先の銀行名・支店名・口座番号・受取人名のいずれかを間違える。

対策：
- [ ] 司法書士から請求書原本を受領（口頭確認のみは不可）
- [ ] 受取人正式名称（個人名 / 法人名 / 屋号）を確認
- [ ] 振込前に**ダブルチェック**（2 名で確認）
- [ ] 振込控えを案件ファイルに保管
- [ ] 振込完了を司法書士に連絡

---

## 7. 借入金額 vs 抵当権設定額の整合確認

**よくある懸案**：借入金額と抵当権設定額が一致しない見え方になる。

理由（説明テンプレート）：
- 抵当権設定額には**保証料・利息・諸費用分の余裕**を含む
- 借入元金より 10〜20% 高く設定されるのが一般的
- 例：借入 4,230 万円 → 抵当権設定 4,930 万円（差額 700 万円は枠）

→ 数字が一致しないことをお客様が不安に思うケース多発。**事前に説明**すること。

---

## 8. 権利証（登記識別情報通知）の取り扱い

- 登記完了後、約 1 ヶ月で発行
- 司法書士から**新居宛に書留**で送付
- お客様への事前案内：
  - 「不在票が入ったら必ず再配達依頼を」
  - 「受領後は紛失しないよう大切に保管」
  - 「将来の売却・担保提供時に必要」
  - 「再発行は原則不可」

## 9. 当日 決済の段取り（時系列）

```
10:00  関係者集合（銀行応接室 or 司法書士事務所）
10:05  本人確認（司法書士が免許証等で確認）
10:15  書類記入・押印
       - 金消契約書
       - 登記委任状
       - 抵当権設定書類
10:30  司法書士が書類完備を確認
10:45  銀行が融資実行（買主口座へ入金）
10:50  買主から売主へ売買代金振込
       買主から司法書士へ費用振込
11:00  司法書士が法務局へ移動 or 当日オンライン申請
11:30  鍵引渡し（即日引渡し案件の場合）
       または引渡し日程の最終確認
12:00  解散
```

⚠️ **当日中に法務局申請を完了**することが原則（翌日になると順位が変わるリスク）。

## 10. 決済後 30 日以内の対応

- [ ] 登記完了確認（司法書士から連絡）
- [ ] 権利証 到着確認
- [ ] 既存口座の住所変更（銀行・カード会社・各種サービス）
- [ ] 引越し完了確認
- [ ] アフターフォロー連絡（入居 2 週間後目安）
- [ ] **保険見直し 初回案内**（[該当マニュアル参照](../周辺サービス/保険見直し初回案内マニュアル.md)）
