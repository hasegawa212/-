## 1. このチェックリストの位置づけ

住宅ローン案件の **事前申込 → 契約 → 本審査 → 金消 → 決済** の 5 工程について、
各段階での **エンド側 / 業務側の必要書類** を網羅した実務リファレンス。

既存マニュアルとの関係：

- 個別の銀行別審査ルール → アプリ「住宅ローン シミュレーター」
- 司法書士との連携詳細 → [司法書士連携運用マニュアル](../外部パートナー/司法書士連携運用マニュアル.md)
- 三為中間省略の書面ルール → [三為中間省略 標準フロー](../仕入れ・案件管理/三為中間省略標準フロー.md)
- 決済日のクリティカルパス → [住宅ローン決済日対応マニュアル](./住宅ローン決済日対応マニュアル.md)

本ドキュメントは **「どの工程で何を準備するか」** の書類チェックリストに特化。

## 2. 工程① 事前審査

> 審査回答期間: **1 週間前後**
> 事前審査承認後、司法書士へ登記依頼を即時起票する

### 2.1 エンド側 必要書類

- [ ] 身分証（顔つき + 保険証の 2 種）
  - 顔つき: 運転免許証 / マイナンバーカード / 住基カード / パスポート
- [ ] 源泉徴収票（直近 2 期分）
- [ ] CIC（個信開示）
  - カード所持者: ネットから取得
  - カード非所持者: 郵送（**約 1 週間かかる**ため早めに着手）
- [ ] 事前申込書
- [ ] 返済表（**他借入がある場合のみ**）
  - 別名: エビデンス
  - 証明書・スクショ等、借入残高がわかるもの

### 2.2 業務側 必要書類

- [ ] **販売マイソク**
  1. アルリットから図面（雛形）を開く
  2. 会社情報をどの会社にするか確認
  3. 仕入れ図面を見ながら必要箇所を入力（**価格はブランク**）
  4. 案件相談シート LINE へ提出（担当にメンション + 価格は上司指示を文面記載）
  5. 業務用アルリットに添付（仕入れ図面・提案図面それぞれ格納）

- [ ] **謄本（建物）**
  1. 物件住所の法務局へ家屋番号を取得
     - 「○○県、○○市○○丁目○○の家屋番号を教えてください」
  2. ネット謄本にログイン（情報保存済み）
  3. 必須事項と家屋番号を入力
  4. PDF 保存

- [ ] **資金計画表**
  - テンプレ: `09_資料・アーカイブ/テンプレート/【資金計画書銀行用】` 参照

## 3. 工程② 契約

### 3.1 エンド側 必要書類

- [ ] 実印
- [ ] 身分証
- [ ] 手付金（金額はお客様により変動）

### 3.2 業務側 必要書類

- [ ] 売買契約書（1 部）
- [ ] 重要事項説明書（2 部：お客様控え + 弊社控え）
- [ ] 手付金領収書
- [ ] 印紙（金額はその都度変動）
- [ ] 謄本（建物 + 土地）

### 3.3 手付金領収証 作成手順

1. アルリットから手付金領収証を開く
2. エンド様にあった会社情報で作成
3. エンド様の氏名・金額・建物詳細を記入（**金額は上司確認**）
4. 印刷
5. 200 円の印紙を貼り、捺印
6. スキャンしてアルリットへ格納

## 4. 工程③ 本審査

> **本審査は基本的に銀行への来店が必須**
> 本審査回答期間: **約 1 週間**
> 本審査時の持ち物は銀行やお客様により変動あり

### 4.1 エンド側 必要書類

- [ ] 実印
- [ ] 銀行印（銀行登録印鑑）
- [ ] 身分証
- [ ] 売買契約書（原本）
- [ ] 重要事項説明書（原本）
- [ ] 手付金領収書（原本）

**公的書類**

- [ ] 住民票（2 部・世帯全員記載・本籍省略・マイナンバーコード省略・現住所）
- [ ] 印鑑証明書（2 部・現住所）
- [ ] 課税証明書（1 部・直近 2 期分）
- [ ] 納税証明書（1 部）

### 4.2 業務側 必要書類（物件資料）

- [ ] 公図
- [ ] 地積測量図
- [ ] 配置図
- [ ] 立面図
- [ ] 平面図
- [ ] 建築確認済証
- [ ] 謄本（建物 + 土地）
- [ ] 間取図
- [ ] 前面道路要約書
- [ ] 資金計画書（諸費用明細）

## 5. 工程④ 金消（金銭消費貸借契約）

### 5.1 エンド側 必要書類

- [ ] 身分証
- [ ] 実印
- [ ] 銀行印（銀行登録印鑑）
- [ ] 通帳

**公的書類**

- [ ] 住民票（3 部）
- [ ] 印鑑証明書（3 部）

### 5.2 業務側 必要書類

- [ ] **火災保険見積書・申込書**
- [ ] 登記費用見積書（→ 司法書士先生シートで作成）
- [ ] 最終金請求書（**石井専務作成**）
- [ ] 固定資産税・都市計画税精算書

### 5.3 火災保険申込手順（あいおいニッセイ）

1. デスクトップに記載の AD1 を開く
2. システムあいおいニッセイを選択
3. 契約手続き → 見積書・申込書 → 新規 → 実行
4. 下記の項目を選択（他はそのまま）
   - 保険種類: **タフ・住まいの保険（2019 年 10 月以降始期契約）**
   - パターン: **3 パターン**
   - 所在地: 対象物件の所在地
   - 申込日: **金消日**
   - 始期日: **決済日**
5. 保険料計算は見本を見ながら入力（赤い修正が出たら計算が出るまで修正）
6. 住所氏名: お客様情報を入力（**保険の対象所在地は購入物件の情報**）
7. 申請書作成（黄色い修正は無視可）

## 7. 三為中間省略時の司法書士連携

> 詳細は [司法書士連携運用マニュアル](../外部パートナー/司法書士連携運用マニュアル.md) 参照

### 7.1 登記名義人の順序

**売主 → MA → 東洋 → 買主**（三為中間省略 連件登記）

- AB 間: 売主 → MA（マーシャルアーツ）
- BC 間: MA → 東洋建設ホーム
- CD 間: 東洋建設ホーム → 買主（エンド）

## 8. 工程横断 留意点

- **CIC は約 1 週間かかる**ため、事前審査ステージ着手と同時に手配
- **本審査は来店必須**、エンド様のスケジュール確保
- 火災保険の **始期日 = 決済日** を必ず合わせる
- 三為案件は **司法書士に登記名義人の順序を明示**

---

## 9. 機微情報の取り扱い

- このチェックリストには **顧客名・物件詳細・口座・金額** を一切含めない
- 個別案件の進捗は別途 [銀行審査状況 パイプライン運用ガイド](../案件管理基盤/銀行審査状況パイプライン運用ガイド.md) のルールで Google Sheets 管理
- 機微情報を含むファイルは git にコミットしない
