## 1. 三為（さんため）中間省略 とは

「**第三者のためにする契約**」を活用した不動産の中間省略登記。

通常の流れ：
```
売主A  →（売買契約 + 所有権移転）→ 当社B  →（売買契約 + 所有権移転）→ 買主C
```

中間省略の流れ：
```
売主A  →（売買契約・第三者のためにする特約）→ 当社B
                                                ↓
                                            買主C を指定
                                                ↓
売主A  →（所有権移転）→ 買主C（A→C 直接移転、B は登記スキップ）
```

→ B は**登記費用・登録免許税・不動産取得税**を負担せずに済む。

---

## 2. 適用ケース

- 当社が AB 間で物件を仕入れ、BC 間で買主に再販するケース
- 短期保有（仕入れ〜再販まで 1 ヶ月以内が標準）
- リフォーム前提販売（リフォームのために一旦仕入れる場合）

---

## 3. 通常契約との切替判断

| 項目 | 通常 AB→BC 別契約 | 三為中間省略 |
| --- | --- | --- |
| 登記回数 | 2 回 | 1 回 |
| B の登録免許税 | 必要 | 不要 |
| B の不動産取得税 | 必要 | 不要 |
| B の保有期間制約 | なし | あり（短期前提） |
| 銀行融資 | 通常 | **銀行確認必須**（一部行で不可） |
| 売主の同意 | 不要 | 必須（特約条項） |
| 買主の理解 | 不要 | 必須（複雑な仕組み） |
| 書類の複雑度 | 標準 | 高い（覚書必要） |

→ **節税効果は大きいが、書類ミスのリスクが高い**。坂本（契約課）の関与必須。

---

## 4. 必要な書類一式

### AB 間（売主×当社）
- 売買契約書（**第三者のためにする特約条項 入り**）
- 重要事項説明書
- 覚書（中間省略の合意確認、買主指定の権利確認）

### BC 間（当社×買主）
- 売買契約書
- 重要事項説明書
- 覚書（中間省略への合意確認）

### 同日決済時
- A→C 所有権移転登記書類
- C への抵当権設定書類
- 司法書士への委任状（A・C 双方）

---

## 5. 標準タイムライン

```
[T-30 日]  AB 間 売買契約締結（中間省略特約入り）
[T-25 日]  BC 間 売買契約締結（中間省略合意覚書付き）
[T-20 日]  銀行への中間省略スキーム説明・確認
[T-14 日]  司法書士アサイン、書類ドラフト確認
[T-7 日]   必要書類リスト最終確認
[T 日]     決済（A→C 同日所有権移転、当社は登記スキップ）
```

## 6. 注意点・落とし穴

### 6-1. 銀行確認
- すべての銀行が三為中間省略に対応するわけではない
- **事前確認必須**：BC 間融資銀行に「三為中間省略案件として進行」を明示
- 銀行が拒否した場合は通常 AB→BC 別契約に切替（コスト増を覚悟）

### 6-2. 売主への説明
- 売主（A）にとっては「自分の名前のまま C に売る形」になる
- 「なぜ B（当社）が間に入るのに B 名義で登記しないのか」と疑問を持たれる
- → **「B が登記費用を節約するため、A→C 直接移転にさせていただきたい」**と正直に説明

### 6-3. 買主への説明
- 買主（C）から「A はどんな人？」と聞かれることがある
- 個人情報の壁があるため、説明範囲を事前整理
- 中間省略の仕組みは買主にも理解してもらう（重要事項説明）

### 6-4. 短期譲渡所得との関係
- B（当社）の利益は**短期譲渡所得**になるため、税率が高い
- ただし三為中間省略により B 側の取得税・登記費用が不要 → トータルで有利
- 税理士に**案件毎に**スキーム妥当性を確認

### 6-5. 住所変更の取り扱い
- 買主（C）が決済までに住所変更する場合、新住所の住民票を**先に提出**してから契約・登記
- 旧住所のまま登記すると、後で住所変更登記が必要になる

## 7. 覚書フォーマット（要旨）

### AB 間 覚書 必須条項
1. 第三者のためにする特約の確認
2. B が第三者（買主 C）を指定する権利
3. C の指定方法と期限
4. 所有権移転は A→C 直接で行う合意
5. 売買代金の支払いフロー（C から A への直接 or B 経由）

### BC 間 覚書 必須条項
1. C が中間省略スキームに合意
2. A→C への直接所有権移転に合意
3. 第三者のためにする契約の受益意思表示
4. 売主（A）の本人確認は司法書士経由で行うことの合意

⚠️ **覚書ドラフトは坂本がレビュー必須**。法務確認も推奨。

---

## 8. 失敗パターン

| ❌ パターン | 影響 | 是正 |
| --- | --- | --- |
| 銀行確認なしで進行 | 決済直前に否決リスク | 事前審査段階で銀行に明示 |
| 覚書の不備 | 登記不能 | 司法書士のリーガルチェック必須 |
| 売主への説明不足 | 不信感・契約破談 | 正直説明 + 理由開示 |
| 住所変更の段取りミス | 登記後の追加手続き | 住民票変更を先行 |
| 同日決済の段取り不足 | 当日トラブル | 司法書士と事前打合せ |

## 9. 案件記録に残すべき情報

各 三為中間省略 案件で、以下を案件ファイルに保管：

- [ ] AB 間 売買契約書 + 特約条項（控え）
- [ ] BC 間 売買契約書 + 合意覚書（控え）
- [ ] 銀行への事前確認記録（メール等）
- [ ] 司法書士の書類チェック記録
- [ ] 売主・買主への説明記録
- [ ] 決済時の同日処理記録
- [ ] 税理士確認記録

→ 次回類似案件のテンプレ参照ソースになる。
