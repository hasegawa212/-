# 物件仕入れマニュアル（再現性版 v1）

**このマニュアルのゴール** 入社して間もない人が、**先輩に聞かずに、1人で「仕入れる物件」を見極め → 業者に問合せできる**状態になること。迷ったら必ずこのマニュアルに戻る。口頭で聞く前に「ここに書いてあるか」を先に探す。

## 0. まず全体像（何のために仕入れるのか）

うちの仕入れは「安く買って高く売る」ためではなく、**「銀行（住宅ローン）が通る物件を、買える個人のお客様に合わせて仕入れる」**仕事。

- 【出口】個人のエンド客が、住宅ローンを組んで買う ← ここから逆算する
- 【仕入れ】客の年収・勤務先・居住エリアに整合する戸建を仕入れる

だから判断の軸は常にこの2つ：

1. **銀行が貸せるか**（担保評価が出るか・客との整合性があるか）
2. **エンド個人客に売れる戸建か**（戸建・新耐震・空室・施工会社）

この2つを外した物件は、どれだけ安くても**仕入れない**。

## 仕入れの全体フロー（今どこにいるか）

| ステップ | やること | これができたら次へ |
| :-- | :-- | :-- |
| ★1 エリア選定 | 客に合うエリアを絞る | 対象エリアが1つ決まった |
| ★2 物件選定 | 条件表でふるいにかける | 条件を全部⭕で通過した |
| ★3 担保評価 | 価格と評価が成立するか確認 | 「幅（利益）」が出ると確認 |
| ★4 問合せ | 業者に電話して条件を詰める | 仕入れ可否の返事をもらう |

新人はまず **★2の条件表を完璧に覚える**のが最優先。ここで弾けるかどうかが8割。

## ★ステップ1：エリア選定

### 手順

1. お客様の「勤務先・現住所・実家」の3つの住所を確認する
2. 下の優先順で、**車40分前後の圏内**でエリアを絞る

| 優先 | 基準エリア | 範囲 |
| :-- | :-- | :-- |
| 第1優先 | 現在勤めている会社の勤務地周辺 | 車40分前後 |
| 第2優先 | 現在の住まい周辺 | 車40分前後 |
| 第3優先 | 実家周辺 | 車40分前後 |

3. 客が支店のある大企業勤務なら、**各支店周辺**も候補に出せる

### 考え方（なぜ40分・なぜ整合性か）

銀行は「この客が、この物件を買うのは自然か？」を見ている。勤務先や生活圏から離れすぎた物件は「なぜそこを買う？」と疑われ、**審査で落ちる**。日本全国どこでも買えるわけではない、という前提を忘れない。

### 判断基準（迷ったら）

- 勤務先・住まい・実家のどれにも40分で当てはまらない → **そのエリアは捨てる**
- 大企業勤務で支店が近くにある → 支店基準で広げてOK

### よくあるミス

- 安い物件を見つけて、客の生活圏を無視してエリアを広げてしまう（→審査落ち）

### 使うツール

- Googleマップ（車での所要時間で確認。直線距離ではなく**車40分**で見る）

## ★ステップ2：物件選定（条件表）

⚠️ **この条件を1つでも満たさない物件は仕入れNG。例外を作らない。**

### 仕入れ物件 条件一覧

| 項目 | ⭕仕入れる | ❌仕入れない／🔺要確認 | なぜ |
| :-- | :-- | :-- | :-- |
| 用途地域 | 市街化区域 ⭕ | 市街化調整区域 ❌／非線引区域 🔺要確認 | 調整区域は原則建築不可で価値が出ない |
| 築年（耐震） | 新耐震＝1981年6月1日以降 ⭕ | 旧耐震＝1981年5月31日以前 ❌ | 旧耐震は評価・ローンが付きにくい。築浅・2000年以降ならなお良し |
| 構造 | 木造・鉄筋など何でも可 ⭕ | － | 構造は評価の足切りにならない |
| 道路 | 公道 ⭕／私道 ⭕ | 私道は使用許可証・覚書の有無を確認 🔺 | 接道義務（幅員4m以上の道路に2m以上接道）を満たすか |
| 施工会社 | 飯田以外 ⭕（大手ならなお良し） | 飯田（飯田HD）❌ 価値無し | 飯田HD物件は担保評価が付きにくい |
| 現状 | 空室 ⭕ | 居住中 🔺 | 居住中は所有者と日程調整が必要で内見しづらい |
| 告知事項 | なし ⭕ | 容積率オーバー ❌／未登記 🔺／仮換地 ❌ | 容積率オーバーは住宅ローンが組めない／未登記は登記が必要 |
| 物件種別 | 戸建 ⭕ | 建売 ❌ | 建売は隣に比較対象があり価格を上げにくい・値段がバレる |
| 価格 | 本物=仕入：1,700〜2,000万 | 充物=出口：2,800〜3,000万 | 本物＝実際に買う額／充物＝客に充てて売る出口価格。差が幅（利益） |

### 優良ハウスメーカー（評価が付きやすい＝歓迎）

注文住宅専門の会社が施工した物件が**最も評価が高い**。

積水ハウス／セキスイハイム／ヘーベルハウス／タマホーム／三菱地所／ダイワハウス／木下工務店／三井ホーム／一条工務店／トヨタホーム／パナソニックホームズ／大成建設／住友不動産

### 絶対NG（覚えるまで貼っておく）

- **飯田HD の物件**（評価が付かない）
- **仮換地の物件**（区画整理中で従前地の代替地。絶対に仕入れない）
- **容積率オーバー**（住宅ローン不可）
- **旧耐震**（1981年5月31日以前）

### 考え方

条件表は「うちが過去にやられた失敗の集合」。1つ妥協すると、**最後の銀行審査か出口で必ず詰まる**。新人のうちは「全部⭕じゃないと進めない」でちょうどいい。🔺が出たら止まって★問合せで確認する。

### よくあるミス

- 🔺（非線引・私道・居住中）を「たぶん大丈夫」で⭕扱いして進めてしまう
- 施工会社を確認せず、後から飯田HDだと判明して時間を無駄にする
- 建売を戸建だと思って拾ってしまう

### 使うツール

- レインズ／スーモ／アットホーム（物件の一次ソース）
- 用途地域・接道は役所または物件資料で確認

## ★ステップ3：担保評価（幅が出るか）

物件を見つけたら、**「銀行評価が出て、利益（幅）が取れるか」**を確認する。

### 手順

1. 対象エリア内の**類似物件**の価格を確認する
2. 下の項目で、選定物件の評価額を見積もる

| 確認項目 | 見るポイント |
| :-- | :-- |
| 敷地面積 | 敷地（土地）面積が近い物件と比較 |
| 建物面積・構造・間取り | 延床面積・構造・間取りが似た物件と比較 |
| 駅距離 | 駅から徒歩何分。遠すぎると評価が下がる |
| 謄本（登記簿） | 前所有者が**いくらで抵当権を設定していたか**を確認 |
| 固定資産税 | 物件・土地の固都税額を確認（評価が付くかの材料） |
| 路線価 | 全国地価マップで路線価を確認 |

### 考え方（路線価とは）

土地に「定価」はなく、用途ごとに複数の価格がある。代表的な2つ：

- **相続税路線価**：国税庁が毎年（1/1時点・7月公表）。公示価格の**約80%**目途
- **固定資産税路線価**：市町村が3年ごとに評価替え。公示価格の**約70%**目途

「幅（利益）」＝ 物件の評価額を算出した上で、本体価格を設定して生まれる差。返済比率（年収に占める年間返済額の割合）から、客が借りられる上限を逆算し、その範囲内で出口価格を決める。

### 判断基準（迷ったら）

- 類似物件より明らかに評価が低い／駅から遠い → 幅が出にくい。**見送り候補**
- 前所有者の抵当権設定額が極端に低い → 評価が出ない可能性。慎重に

### よくあるミス

- 「安いから利益が出る」と思い込み、評価額の裏取り（謄本・路線価・固都税）をしない
- 返済比率を計算せず、客が借りられない価格で出口を設定してしまう

### 使うツール

- 全国地価マップ（路線価）／登記簿謄本／金融電卓（返済比率の算出）

## ★ステップ4：問合せ（業者への電話）

### 手順

1. まず**販売形態**を確認：問合せ先が「売主」か「売主仲介」か
   - 売主自身なら交渉しやすい
   - 他人物（現所有者が別）の場合は、所有者への確認が必要で時間がかかる
2. 下のリストを**順番に**確認する

### 問合せ時の確認リスト（このまま聞く）

- 価格を乗せて、**最終的に個人のエンド客へ売却**して良いか
- 契約から引渡し（決済）まで**3ヶ月のサイト（猶予）**は可能か
- **同日の連件決済**は可能か（業者・個人によりNGあり）
- 売主が**売却する理由**（住宅ローン審査で書類が要るため事前確認）
- **告知事項**の有無（下記事例）
- エコ給湯設備の有無（優良住宅だと金利優遇・評価UPの可能性）
- 町内会費の有無（エリアにより発生）
- 施工会社（評価に直結。飯田HDは✕）
- **手付金の額**（相場50〜100万。安い方が解除時に損しない）

#### 告知事項の事例（あれば要注意）

①事件・事故があった ②建物の耐久性に問題 ③雨漏り・基礎のひび割れ ④不動産価値の下落 ⑤保険・ローン適用への影響

### 考え方

問合せの目的は「いい物件か確認する」だけでなく、**うちの売り方（価格を乗せて連件決済でエンドに売る）が成立する物件か**を確かめること。だから「売却OKか」「サイト3ヶ月」「連件決済」は最優先で押さえる。

### 判断基準（迷ったら）

- 「価格を乗せてエンドに売るのはNG」と言われた → **その物件は撤退**
- サイト3ヶ月／連件決済がどちらも取れない → 仕入れ難易度が高い。先輩に共有

### よくあるミス

- 物件のスペックばかり聞いて、**売り方の成立条件（売却OK・サイト・連件）を聞き忘れる**
- 告知事項・売却理由を聞かず、審査直前で書類が揃わず止まる

### ★電話トークスクリプト（新人はこのまま読む）

電話は会社の顔。あなたが会社の印象になる。礼儀は「これでもか」というくらい丁寧に。業者は担当によって対応が悪い人もいる。**いちいちイライラしない**（疲れるだけ）。

**① 名乗り**
「お世話になります。株式会社○○の△△と申します。○○（媒体）に掲載されている【物件名／所在地】の件でお電話いたしました。ご担当者様はいらっしゃいますか。」

**② 用件**
「こちらの物件、当社で価格を乗せて、最終的に個人のお客様へ住宅ローンでご購入いただく形を検討しております。いくつか確認させてください。」

**③ 確認（上のリストを順に）**
「まず、価格を乗せてエンドのお客様へ売却することは可能でしょうか？」「契約から決済まで3ヶ月ほどのサイトはいただけますか？」「同日の連件決済は対応可能でしょうか？」…（以下リスト順）

**④「契約してからじゃないと（教えられない／進められない）」と言われた場合**
「承知しました。ただ、契約前に住宅ローンの審査を進めておかないと、引渡しまでに時間がかかってしまう可能性があります。先に◯◯だけご確認いただくことは可能でしょうか。」

**⑤ 締め**
「ありがとうございます。では◯◯の点、ご確認のうえご連絡いただけますと幸いです。よろしくお願いいたします。」

## 用語集（新人がつまずく言葉）

| 用語 | 意味 |
| :-- | :-- |
| レインズ（REINS） | 国交省指定の不動産流通機構が運営する物件情報ネットワーク。業界インフラ |
| アットホーム | 全国の加盟店物件を集めた不動産情報サイト |
| 返済比率 | 年収に占める年間返済額の割合。銀行が「返せる計画か」を判断する指標 |
| 担保評価（担保評価額） | その物件を担保にしたとき、銀行が認める価値の金額。融資可能額の基準 |
| 路線価 | 土地評価の公的価格。相続税路線価（公示の約80%）と固定資産税路線価（約70%）がある |
| 幅（利益） | 物件評価額をもとに本体価格を設定して生まれる差益 |
| 通常決済 | 売主→買主の2者間で所有権移転 |
| 連件決済 | 2者以上が同日・同時に複数の決済（所有権移転）を行う |
| 三為決済 | 3者間決済。売主から（最終）買主へ直接所有権を移転する形 |
| サイト | 契約から決済（引渡し）までの猶予期間。うちは3ヶ月が目安 |
| 接道義務 | 都市計画区域内で建物を建てる際、幅員4m以上の道路に2m以上接した敷地である必要 |
| 仮換地 | 区画整理事業中に、従前の宅地の代わりに使用指定された土地。**仕入れNG** |
| 本物 | 実際に仕入れて売る物件。仕入価格帯1,700〜2,000万 |
| 充物（あて物） | 客に充てて売る出口価格帯2,800〜3,000万。本物との差が幅（利益） |

## ⚠️ 要設定（メイン行の数字で確定する）

1. 「本物／充物」の定義 → **確定済**（本物＝仕入1,700〜2,000万／充物＝出口2,800〜3,000万）
2. **返済比率の審査金利・上限%** — フラット35基準(30%/35%)で記載済。うちのメイン行の実数値に置き換えて固定する
3. **非線引区域🔺の確認** — 役所(都市計画課)で用途地域指定・建築可否・開発許可を確認（記載済）
4. **私道🔺の覚書** — 通行・掘削の承諾書と持分を確認。無ければ撤退寄り（記載済）

## 判定フロー（見た瞬間に上から切る）

1つでも❌が出たらそこで終了。

1. **施工会社** → 飯田HD なら ❌即終了
2. **物件種別** → 建売 なら ❌即終了
3. **用途地域** → 調整区域❌／非線引🔺役所確認
4. **築年** → 旧耐震(1981/5/31以前) ❌即終了
5. **告知/権利** → 容積率オーバー❌／仮換地❌／未登記🔺
6. **道路** → 私道で覚書なし 🔺(撤退寄り)
7. **現状** → 居住中 🔺(内見調整)
8. **エリア整合** → 勤務/住まい/実家から車40分外 ❌
9. **価格帯** → 本物1,700〜2,000万で仕入→充物2,800〜3,000万で出口が成立するか

→ **1〜9全通過なら 担保評価 → 問合せ へ**

## 仕入れ判定シート（物件1件ごとに記入→案件相談シートに貼る）

新人が物件を見たらこのシートを埋める。埋まらない＝確認漏れ。

- **物件情報**：物件名/所在地・媒体・売出価格・売主or売主仲介・担当/連絡先・記入者/日付
- **条件チェック**：上の9項目を ⭕/❌/🔺 で記入
- **担保評価メモ**：類似物件価格・謄本(前抵当権設定額)・路線価/固都税・想定出口(充物)・幅の見込み
- **問合せ結果**：エンド売却OK?／サイト3ヶ月OK?／連件決済OK?／売却理由／手付金
- **総合判定**：仕入れGO / 見送り / 要相談 ＋ 理由
