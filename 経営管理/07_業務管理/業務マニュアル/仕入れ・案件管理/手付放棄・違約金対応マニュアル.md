# 手付放棄・違約金対応 標準フロー v0.1

> 作成日: 2026-05-16
> 主担当: ★ 未指名（営業責任者 + 法務確認役）
> バックアップ担当: ★ 未指名
> 元情報: 過去発生案件の構造抽出
> 脱属人化スキーム該当原則: ⑤意思決定基準書、⑥経営層 No.2 への権限委譲

---

## 1. このマニュアルの対象シナリオ

買主が以下の状態になった案件：

- A) **連絡が取れなくなった**（着信拒否・LINE 既読つかず・架電不通）
- B) **手付放棄の意思表示**を行った
- C) **白紙撤回を申し出**たが理由が契約解除要件に該当しない
- D) **消費者センター・弁護士等 第三者経由**で連絡が来た

→ どのシナリオも、放置すると**契約上の期限到来 → 違約金請求 / 仲介手数料の取り扱い**を社内判断する必要が発生。

---

## 2. 全体フロー（標準）

```
[1] 状況把握・記録
   ↓
[2] 第一次接触試行（電話・LINE・郵送）
   ↓
[3] 社内会議で対応方針決定
   ├─ 手付放棄で進行
   ├─ 違約金請求
   └─ 仲介手数料の扱い決定
   ↓
[4] 書面対応（A B 間 / B C 間）
   ↓
[5] 期限管理（最終期限まで待機）
   ↓
[6] 期限到来時の最終判断
   ├─ 違約金請求実行
   ├─ 仲介手数料回収
   └─ 案件クローズ・記録保管
```

---

## 3. ステップ詳細

### Step 1：状況把握・記録

直ちに以下を CRM / 案件ファイルに記録：

- [ ] 最後にお客様と接触した日時・手段
- [ ] お客様の最後の発言・意思表示
- [ ] 連絡が取れなくなった経緯（何がきっかけか）
- [ ] 第三者からの連絡があった場合：相手の所属・連絡先・主張内容
- [ ] 契約締結日・手付金額・決済予定日
- [ ] 売主側の状況（売主にどこまで伝えているか）

⚠️ **記録は時系列で残す**。後の違約金請求・訴訟対応で証拠になる。

### Step 2：第一次接触試行

**順序を守る**（後で「努力しなかった」と言われないため）：

1. 電話：3 日連続、朝・昼・夜の異なる時間帯で
2. SMS：「ご連絡いただけませんか」程度の柔らかい文面で
3. LINE：既読確認まで
4. メール：契約時の連絡先へ
5. **書留郵便**：契約書類記載住所へ「ご連絡のお願い」
6. ご家族・勤務先連絡先（契約時取得済の場合）

→ Step 5 の書留送付までやって反応がなければ、**「連絡可能な手段は尽くした」と社内記録**。

### Step 3：社内会議で対応方針決定

仕入れ会議または臨時会議で以下を合議：

| 判断項目 | 選択肢 | 判断基準 |
| --- | --- | --- |
| AB 間（売主×当社）の扱い | a) 手付放棄で進行 / b) 契約継続交渉 / c) 違約金請求 | 売主の意向、当社の損害 |
| BC 間（当社×買主）の扱い | a) 手付返還なし / b) 一部返還 / c) 違約金請求 | 買主の事情、信用情報 |
| 仲介手数料の扱い | a) 請求しない / b) 半額請求 / c) 全額請求 | 業務遂行段階、契約条項 |
| 消費者センター対応 | a) 静観 / b) 当社見解書面送付 | 第三者介入の度合い |

**判断は必ず複数名で**（脱属人化原則 ⑤）。1 人判断は避ける。

### Step 4：書面対応

合議結果に基づき、以下を書面化して**書留 + 控え保管**：

- AB 間：売主への状況報告書、手付放棄確認書
- BC 間：買主への解約確認書（連絡が取れない場合は配達証明付き）
- 社内：判断理由書（誰が、いつ、何を、なぜ判断したか）

### Step 5：期限管理

- 売買契約書に記載された「契約解除可能期限」「決済期限」を確認
- カレンダーに**最終期限の 7 日前にアラート**
- 期限までは「本人からの連絡待ち」の状態を維持
- 期限到来時に Step 6 へ

### Step 6：期限到来時の最終判断

| 状況 | アクション |
| --- | --- |
| 期限内に連絡があった | 個別対応（再合議） |
| 期限内に連絡なし、手付放棄成立 | AB 間決済進行 / BC 間クローズ |
| 期限内に連絡なし、違約金請求方針 | 内容証明送付 → 反応 7 日待ち → 弁護士相談 |
| 仲介手数料回収方針 | 請求書送付 → 反応 14 日待ち → 法的措置検討 |

## 4. 第三者介入時の対応（消費者センター等）

### 基本姿勢
- **本人不在のまま第三者と本契約の話を進めない**（個人情報保護・委任関係不明確）
- 消費者センターは「間に入る」立場ではなく「アドバイス機関」であることが多い → **最終的に本人からの直接連絡を要請**
- 弁護士が代理人として連絡してきた場合は、**委任状の提示**を求めてから対応

### 対応スクリプト例

> 「ご連絡ありがとうございます。本件は当社とご本人様との契約事項ですので、最終的なお話はご本人様から直接いただきたく存じます。ご本人様にその旨お伝えいただけますでしょうか」

## 5. 仲介手数料の判断基準

仲介手数料は**「業務をどこまで遂行したか」で按分判断**：

| 業務段階 | 請求妥当性 |
| --- | --- |
| お客様紹介・初回面談のみ | 請求困難 |
| 物件提案・内見実施 | 請求困難 |
| 事前審査・契約書準備 | 一部請求検討 |
| 売買契約締結 | **原則全額請求可** |
| 本審査・本承認 | 全額請求可 |
| 金消・決済直前 | 全額請求可 + 諸経費請求検討 |

→ **売買契約締結後**であれば仲介手数料請求の根拠が立つ。  
→ 締結前であれば請求は実務上困難（時間労力の損失として記録のみ）。

---

## 6. 違約金請求の判断基準

| 判断材料 | 請求する | 請求しない |
| --- | --- | --- |
| 売主側の損害額 | 大きい | 軽微 |
| 買主の支払能力 | あり（属性確認済） | なし（収入証明等） |
| 訴訟費用対効果 | 違約金額 > 訴訟費用 | 逆 |
| レピュテーションリスク | 低い | 高い（SNS 等） |
| 過去の関係性 | なし | 紹介者からの紹介等 |

→ **「請求できる」と「請求すべき」は別**。経営判断として総合評価。

## 7. 記録・証跡管理

以下をすべて**案件ファイル**に時系列保管（最低 5 年）：

- [ ] 通話記録（日時・相手・要点）
- [ ] 送信メール・SMS・LINE スクリーンショット
- [ ] 書留・配達証明の控え
- [ ] 第三者からの連絡記録
- [ ] 社内会議議事録（判断理由含む）
- [ ] 書面（手付放棄確認書、内容証明等）
- [ ] 入金記録（手付金・違約金等）

⚠️ **個人情報を含むため git には保存しない**（Google Drive 限定共有領域）。

---

## 8. やってはいけないこと

| ❌ | 理由 |
| --- | --- |
| 1 人で判断・実行 | 後で会社責任になる |
| 接触試行を省略 | 「努力義務」を果たしていない |
| 第三者と本契約の交渉 | 委任関係不明確 |
| 感情的な書面送付 | 後の訴訟で不利な証拠化 |
| 期限を放置 | 違約金請求機会の喪失 |
| 記録を残さない | 紛争時に立証困難 |
| 売主への報告遅延 | 売主の信頼喪失 → 仕入れルートの毀損 |

## 9. エスカレーション基準

以下が発生した時点で**経営層 No.2 / 経営層 No.1 にエスカレーション**：

- 弁護士が代理人として接触してきた
- 消費者センター以外の第三者機関（業界団体・行政等）が動いた
- 違約金請求額が 100 万円超
- SNS・口コミ等で当社名が出た
- 売主側との関係悪化リスクが見えた
